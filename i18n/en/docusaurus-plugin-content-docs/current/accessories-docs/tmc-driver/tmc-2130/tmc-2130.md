## Product Overview

The TMC 2130 is a high-performance two-phase stepper motor driver IC featuring standard SPI and step/dir interfaces. It offers low noise and ensures no step loss even under high-load conditions.

## Specifications

| Specifications | TMC 2130 |
| -------------- | -------- |
| Driver IC      | TMC 2130 |
| Input voltage  | 24V      |
| Drive current  | 2A peak  |
