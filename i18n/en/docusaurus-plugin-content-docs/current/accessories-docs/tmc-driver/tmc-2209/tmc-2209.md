## Product Overview

The TMC2209 is an ultra-quiet motor driver designed for two-phase stepper motors. It features a continuous drive current of 2 A and a peak current of 2.8 A. Compared to the TMC2208, this integrated circuit not only offers a 0.6 A to 0.8 A increase in drive current but also includes a motor lock test function.

## Specifications

| Specifications  | TMC 2209          |
| --------------- | ----------------- |
| Driver IC       | TMC 2209          |

| Drive current   | 2A RMS, 2.8A peak |

## Hardware Configuration

### Configuring Current in Step/Dir Mode

Trinamic Driver Current Calculation Formula

$I_{RMS}=\frac{325mV}{R_{SENSE}+20mΩ}*\frac{1}{\sqrt{2}}*\frac{V_{VREF}}{2.5V}$	

Using the TMC 2209 Driver Module $110 mΩ _{RSENSE}$ Resistor
		
$I_{RMS}=\frac{325mV}{110mΩ+20mΩ}*\frac{1}{\sqrt{2}}*\frac{V_{VREF}}{2.5V}$			
		
This can be simplified to

$I_{RMS}=\frac{V_{REF}}{\sqrt{2}}$

Calculate the peak current

$I_{PEAK}=I_{RMS}*\sqrt{2}=V_{REF}$

:::note
$V_{REF}$ Range: 0.2V – 2.2V

Default voltage Vref: 1.2V +/- 0.1V

Default current RMS: ~0.9A 
:::

### UART Mode Pin Configuration

By default, the UART pin is connected to Pin 4, which is the PDN_UART pin shown on the left in the figure below. If you wish to use Pin 5 as the UART pin, you must remove the resistor and solder it to the two pads shown on the right in the figure below. The PDN must be soldered to the corresponding UART mode selection pin.
