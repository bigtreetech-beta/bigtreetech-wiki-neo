## 产品简介

TMC5160 是一款大功率步进电机驱动控制芯片，使用外置MOSFET，最大电流可达20A，工作时发热小。

## Specifications

| Specifications | TMC 5160  |
| -------------- | --------- |
| 驱动芯片       | TMC 5160  |
| 输入电压       | 8-40V     |
| 驱动电流       | 4.4A peak |

## 硬件配置

### 驱动电流

TMC 5160 的最大驱动电流取决于使用的采样电阻。默认值为 $0.075 \Omega$

| $R_{sense}$ [$\Omega$] | RMS current [A] | Sine wave peak current [A] |
| ---------------------- | --------------- | -------------------------- |
| 0.22                   | 1.1             | 1.5                        |
| 0.15                   | 1.6             | 2.2                        |
| 0.12                   | 2.0             | 2.8                        |
| 0.10                   | 2.3             | 3.3                        |
| 0.075                  | 3.1             | 4.4                        |
| 0.066                  | 3.5             | 5.0                        |
| 0.050                  | 4.7             | 6.6                        |
| 0.033                  | 7.1             | 10.0                       |
| 0.022                  | 10.6            | 15.0                       |

:::warning[更换采样电阻]

如果需要更大电流需要更换采样电阻。由于模块体积限制更换低于 $0.05 \Omega$ 的采样电阻可能会导致驱动损坏。

:::
