## Product Overview

The TMC2240 is a smart, high-performance stepper motor driver IC equipped with serial communication interfaces (SPI, UART) and comprehensive diagnostic functions.

## Specifications

| Specifications | TMC 2240           |
| -------------- | ------------------ |
| 驱动芯片       | TMC 2240           |
| 最大电压       | 8V-36V DC          |
| 驱动电流       | 2.1A RMS 3.0A peak |

## Klipper SPI Config

```klipper_cfg title="printer.cfg"
[tmc2240 stepper_x]
cs_pin: PA1
spi_software_sclk_pin: PA2
spi_software_mosi_pin: PA3
spi_software_miso_pin: PA4
run_current: 0.80
interpolate: True
stealthchop_threshold: 0
```
