## 产品简介

TMC 2130 是一款高性能两相步进电机驱动芯片，具有标准SPI接口和step/dir接口。具有低噪音以及高负载驱动器不丢步。

## Specifications

| Specifications | TMC 2130 |
| -------------- | -------- |
| 驱动芯片       | TMC 2130 |
| 输入电压       | 24V      |
| 驱动电流       | 2A peak  |
