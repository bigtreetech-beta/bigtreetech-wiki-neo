## Product Introduction

The Panda Den H2 is a heavy-duty steel workstation designed for the Bambu Lab H2/A/P/X series. As the foundation of the BIQU Panda-Verse, it integrates a waste collector and tool drawer directly beneath your machine to reclaim desk space. Optional RGB lighting adds personalized style and real-time status feedback, while printable add-ons - including a build plate rack, filament spool holder, and hotend storage - let you fully customize your ultimate printing hub.

## Product Features

- **Desk-Space Reclamation**: Positioned directly beneath your Bambu Lab printer, this heavy-duty steel workstation maximizes efficiency without taking up extra desk space.

- **Integrated Storage & Modularity**: Features a high-capacity waste collector and a dedicated drawer for essential tools like scrapers, pliers, and liquid glue. This modular foundation supports an array of printable add-ons, including a **build plate rack**, **filament spool holder**, and **hotend storage**.

- **Optional Lighting Kit**: Provides personalized style and real-time printer status feedback. Lighting effects can be easily managed via a wireless Web UI.

- **Industrial-Grade Build**: Constructed from high-strength, seamless steel with integrated soft-close dampers for a smooth, premium drawer experience and lasting durability.

## Installation Guide

###  Bambu Lab H2 Series

- Install the Filament Spool Holder Bracket.

    Retrieve the Filament Spool Holder Bracket Hardware from the drawer. Use the two included **M4x5 button head screws** to secure it to the mounting holes at the rear of the main body as shown.

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-9.webp').default} width="70%"/>

- Install the Spool Holder.

    Secure the spool holder included with the Bambu Lab H2 series onto the bracket as shown.

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-10.webp').default} width="70%"/>

- Print the Sealing Covers.

    Since the Panda Den H2 is designed to be compatible with Bambu Lab H2/P/X/A series machines, there are two openings on the top: one for the A-series waste outlet and one for the H2/P/X-series waste outlet. When using H2/P/X series machines, you can use a printed part to seal the A-series waste outlet. Download link: [A-Series Sealing Cover (Label 1 in diagram)](https://makerworld.com/zh/models/2574229-panda_den_air_panda-den-print#profileId-2838353)

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-20.webp').default} width="70%"/>

    Installation Reference

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-21.webp').default} width="70%"/>

- Place the printer directly on top of the Panda Den H2, ensuring the front door of the machine is aligned with the front of the Panda Den H2.

- Assemble the Poop Chute Extension.

    First, print the Poop Chute Extension compatible with the Bambu Lab H2 series: [Click to Download the Files](https://makerworld.com/zh/models/2574246-panda-den-h2-related-3d-printing-files#profileId-2838362)

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-22.webp').default} width="70%"/>

- Next, secure the magnets from the accessory kit to the H2-series poop chute extension mounting holes using six **M3x6 countersunk screws**.

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-12.webp').default} width="70%"/>

- Finally, align the poop chute extension with the waste outlet and magnetically attach it to the rear.

    <ImageView src={require('@site/docs/panda-series/module/panda-den-h2/img/panda-den-h2-11.webp').default} width="70%"/>
