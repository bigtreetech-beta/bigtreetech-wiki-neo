## Product Overview

The Panda Status P2 is a dedicated upgrade accessory for P2S/X2D printers, bringing H-series-style status display capabilities to your machine. It offers customizable status colors, built-in audio pickup for responsive lighting effects, and an integrated light strip and control board design that makes installation cleaner and cable routing simpler. Wireless configuration, light control, mobile app integration, and OTA firmware updates are either supported or reserved for future use.

## Initial Setup Guide

:::tip

Upon initial powering on or after a factory reset, you will see a blue flowing light effect, indicating that the Panda Status P2 is not yet bound to a printer.

:::

### Wi-Fi Connection Guide

- Connect to the Device Wi-Fi Hotspot

    - Connect to the AP hotspot generated by the device: `Panda Status P2`. The default password is `987654321`.

        <ImageView src={require('./img/panda-status-p2-connect-to-ap-en.webp').default} width="60%" />

- Access the AP Web UI

    - Once connected, PC users can enter the default IP in a browser: `192.168.254.1`. Mobile users should see the Web UI pop up automatically; otherwise, enter `192.168.254.1` in a mobile browser. (Note: This is the factory default IP).

## User Interface Description

### Wi-Fi Interface

<ImageView src={require('./img/panda-status-p2-wifi-binding-interface-description-en.webp').default} width="45%" />

1. WiFi Page Selection: When you tap the icon, it will turn red to indicate that you are on the WiFi page.

2. Current Wi-Fi connection status.

3. Address troubleshooting prompt. If the IP text box is empty, ensure the status shows "Connected." If "Not Connected," please reconfigure network settings.

4. IP Address display (used to access the Web UI from other devices on the same network).

5. Network configuration button. Tap to return to the Wi-Fi Connection Interface

6. Hostname display. You can access the Web UI via the hostname. Default: PandaStatus. If you forget the IP, enter the hostname followed by .local (e.g., http://PandaStatus.local). Tap the text box to edit and "Set Hostname" to confirm.

7. Confirm hostname change.

### AP Interface

<ImageView src={require('./img/panda-status-p2-wifi-binding-interface-description-ap-en.webp').default} width="45%" />

1. AP Page Selection: When you tap the icon, it will turn red to indicate that you are on the AP page.

2. AP Hotspot toggle (On/Off).

3. AP SSID (Hotspot Name). Default:  `Panda_Status_P2_XXXXXXXXXX`

4. AP Password. Default: `987654321`

5. Hotspot IP (Default: 192.168.254.1). If your router uses the same 192.168.254.xxx range, a conflict will occur, preventing the device from binding to the printer. Please manually change the Hotspot IP to a different subnet.

6. Confirm and Restart. The device will reboot to apply changes.

### Settings Interface

<ImageView src={require('./img/panda-status-p2-wifi-binding-interface-description-set-en.webp').default} width="45%" />

1. Settings Page Selection: When you tap the icon, it will turn red to indicate that you are on the Settings page.

2. Language toggle (Default: English).

3. Current Firmware version.

4. Firmware OTA update. Select a `.bin` file.

    - To perform an OTA, tap "Select .bin file" and choose the appropriate file.

        <ImageView src={require('@site/docs/panda-series/module/panda-status-p2/img/panda-status-p2-wifi-binding-ota.webp').default} width="45%" />

    - Wait for the progress bar. The device will automatically restart upon completion.

        <ImageView src={require('./img/panda-status-p2-wifi-binding-ota-1-en.webp').default} width="45%" />

        <ImageView src={require('./img/panda-status-p2-wifi-binding-ota-2-en.webp').default} width="45%" />

5. Factory Reset confirmation button.
