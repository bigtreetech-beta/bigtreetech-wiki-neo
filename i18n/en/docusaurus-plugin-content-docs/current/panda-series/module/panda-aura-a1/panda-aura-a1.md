# Panda Aura A1


## 用户首次连接指南

<a id="first-connection"></a>

:::warning
Panda Aura A1、Panda Aura A1mini、Panda Aura A1 Plus和Panda Aura A1 RGBW所用控制盒软件功能操作步骤一样，以下统称为Panda_Aura_A1，不再区分具体型号。
:::

:::tip
首次连接电源后或执行“恢复出厂设置”命令后，您将看到蓝色跑马灯，代表此时Panda_Aura_A1没有绑定打印机。
:::

### WiFi连接指南

<a id="ap-connect"></a>

- 连接设备WiFi_AP热点
  
  - 连接上Panda_Aura_A1发出来的AP热点: `Panda_Aura_A1_XXXXXXXXXX`. 默认密码是 `987654321`.

    <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step0.webp').default} width="60%"/>

- 访问AP网页

    - 连接上AP后，电脑端可以在浏览器输入默认的ip：`192.168.254.1`;手机端会自动弹出Panda_Aura_A1的Web Ui,也可以在浏览器输入ip：`192.168.254.1`来访问网页。(注：此ip为出厂默认ip)

- 选择语言(默认英文)

    - 选择语言后可点击下一步

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step1.webp').default} width="45%"/>

<a id="wifi-connect"></a>

- WiFi连接界面

    - 首次出厂或恢复出厂设置后，进入此页面会自动扫描周围WiFi，或者手动点击扫描。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step2.webp').default} width="45%"/>

    - 扫描完成后会弹出`扫描完成`弹窗提示，点击OK关闭提示。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step3.webp').default} width="45%"/>

    - 点击Wi-Fi下方的选项，选择想要连接的WiFi

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step4.webp').default} width="45%"/>

    - 输入密码，可点击密码框内右侧的`眼睛`图标来显示输入的密码

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step5.webp').default} width="45%"/>

    - 点击连接，界面会自动跳转到操作界面中的`WiFi`界面,并显示WiFi连接中

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step6.webp').default} width="45%"/>

    - 等待连接一段时间后，会弹出WiFi连接的状态弹窗

        - 连接失败:

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step7.webp').default} width="60%"/>

        - 连接成功，可选择点击[绑定打印机](#printer-bind)跳转到`打印机界面`，也可点击`取消`，继续待在`WiFi界面`

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step8.webp').default} width="60%"/>

### 绑定打印机

- 功能介绍

    <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step9.webp').default} width="45%"/>

    - 1.选择打印机界面，点击后聚焦为红色图标，跳转至打印机界面

    - 2.扫描周围同一局域网下的打印机

    - 3.选择扫描到的打印机

    - 4.输入打印机的SN码

    - 5.输入打印机的访问码

    - 6.输入打印机的IP

    - 7.确定绑定此打印机

- 如何绑定打印机

    - 点击扫描一段时间后，会出现`扫描成功`的弹窗

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step10.webp').default} width="60%"/>

    - 选择扫描出的打印机，可以看到选择打印机后，会自动输入打印机的SN码和IP

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step11.webp').default} width="45%"/>

    - 输入打印机的`访问码`

        * 在打印机上找到`访问码` (这里以 A1 系列打印机为例)

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step12.webp').default} width="60%"/>

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step13.webp').default} width="60%"/>

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step14.webp').default} width="60%"/>

        * 将 `访问码` 输入到 `访问码` 框内,点击绑定即可开始绑定打印机。

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step15.webp').default} width="45%"/><br/>

## 用户界面说明

### WiFi界面

<img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step17.webp').default} width="45%"/>

- 1.WiFi界面选择，点击后聚焦为红色图标，跳转至WiFi界面。

- 2.当前WiFi的连接状态。

- 3.IP地址的相关问题弹窗提示，如果 IP 地址 文本框未显示 IP, 请确保Panda_Aura_A1网络状态显示为 连接成功。如果显示为 未连接, 请配置网络设置。

- 4.IP地址显示(在同一局域网下，浏览器输入此IP也可访问Web网页)。

- 5.WiFi网络配置按钮，点击可跳转到 [WiFi连接界面](#wifi-connect)

- 6.主机名显示，您可以使用主机名访问 Web UI。默认主机名为: Panda_Aura_A1_xxx。如果您不记得 IP 地址，可以在浏览器地址栏中输入主机名加上 .local 访问 Web UI (例如: http://PandaAuraA1.local)。点击文本框可以编辑主机名, 点击 设置主机名 按钮确认修改

- 7.确认修改主机名

### AP界面

<img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step18.webp').default} width="45%"/>

- 1.AP界面选择，点击后聚焦为红色图标，跳转至WiFi界面

- 2.AP热点的开关

- 3.AP热点名称，默认为 `Panda_Aura_A1_XXXXXXXXXX`

- 4.AP热点密码，默认为 `987654321`

- 5.热点IP(默认的热点 IP 为: 192.168.254.1, 如果您自己路由器的 IP 号段也为 192.168.254.xxx, Panda_Aura_A1 与路由器的 IP 号段相同就会产生冲突, 并且导致 Panda_Aura_A1 无法正常绑定到打印机, 请手动修改 Panda_Aura_A1 的热点 IP 使其与路由器使用不同的 IP 号段)

- 6.点击确认会重启设备，重启完成后可连接ap热点。([如何连接ap热点](#ap-connect))

### 设置界面

<img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step19.webp').default} width="45%"/>

- 1.设置界面选择，点击后聚焦为红色图标，跳转至设置界面。

- 2.选择网页中英文(默认英文)。

- 3.显示当前的音效固件版本。

- 4.固件OTA更新选择`.img`文件。

- 5.显示当前的固件版本。

- 6.固件OTA更新选择`.bin`文件。

- 7.恢复出厂设置。

    - 点击`选择.bin文件`按钮，选择需要ota的bin文件。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step20.webp').default} width="45%"/>

    - 等待ota进度，完成后弹窗自动重启。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step21.webp').default} width="45%"/><br/>

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step22.webp').default} width="45%"/>

- 8.点击后，按下确定 [恢复出厂设置](#first-connection)

### 控制面板界面

- 功能介绍

    <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step23.webp').default} width="45%"/>

    - 1.控制面板界面选择，点击后聚焦为红色图标，跳转至控制面板界面

    - 2.当前设备名称

    - 3.单击“切换”以打开和关闭RGB灯光效果

    - 4.打印机库存灯亮起后，自动打开和关闭RGB效果

    - 5.当打印机处于错误状态时，单击开关以覆盖红色闪烁警告灯

    - 7.选择照明效果和当前效果

    - 7.灯光颜色设置

    - 8.亮度

    - 9.速度

- 设备灯效模式

    <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step24.webp').default} width="45%"/>

    - 1-6. 选择后，相应的灯光效果将直接显示。

    - 7.光线的颜色可以根据打印机的温度进行调整。您可以在“安全效果”和“危险效果”下选择两种照明效果，静态或闪光。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step25.webp').default} width="45%"/>

    - 点击颜色设置选项，您将被引导到颜色选择页面。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step27.webp').default} width="45%"/>

    - 选择想要的颜色。

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step28.webp').default} width="45%"/>

        - 1.出厂默认预设颜色。

        - 2.自定义颜色。点击后会出现一个取色盘，可以设置此自定义颜色的颜色值

            <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step29.webp').default} width="60%"/>

        <img src={require('@site/docs/panda-series/module/panda-aura-a1/img/panda_aura_a1_step30.webp').default} width="45%"/><br/>
