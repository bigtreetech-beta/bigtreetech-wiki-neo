## Important firmware update

**Before using Panda Breath, please make sure the firmware has been updated to [V1.0.2 or later](#v102)**

:::warning

This firmware adds some important new features as well as bug fixes. Without installing it, there is the possibility that your Panda Breath may turn off after 30s of heating each time you try to use it as a result of the integrated software safety monitor being overly sensitive on one of the thermistor inputs. This firmware adds a small self-calibration feature on the very first startup which helps to identify the specific characteristics of your Panda Breath.

:::

## Introduction

- The BIQU Panda Breath is a chamber heater and filter for Bambu Lab printers. It uses a 300W PTC heater to warm the chamber, which helps prevent warping, strengthens layer adhesion in engineering materials (ABS, ASA, PC, PA...), and improves first-layer adhesion for all filaments.

- Its dual HEPA and carbon filters remove fine particles and VOCs from the air during printing, reducing odors and improving air quality.

</div>

## Key Features

- Easy Firmware Updates: Includes quick-flash interface for DIY firmware burning and future upgrades.

- Main Power Switch: Convenient one-touch total power cutoff.

- Quick-Access Controls: Dedicated function buttons for streamlined operation.

- Automatic Start/Stop: Intelligently manages chamber heating and air filtration by monitoring the print bed's temperature in real time.

- High-Temperature Capability: Reaches up to 60$^\circ\mathrm{C}$ chamber temperature, ideal for engineering-grade filaments.

- Wireless Control: Onboard ESP32 WiFi module enables remote operation.

- Rapid Heating: High-power PTC heater ensures fast temperature ramp-up.

- Wide Voltage Compatibility: Broad input voltage range maximizes printer compatibility.

- Shared Cable Design: Includes an adapter cable, allowing the device to share one cable with the printer without separate wiring.

    Use your 3D printer to print the Riser. [Download print files here](https://github.com/bigtreetech/Panda_Breath/tree/master/3D%20Model/Mounting%20solution).

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/x-install01-zh.webp').default} alt="" width="35%"  />

- Snapmaker U1:

    Print the Left and Right Mounting Brackets using your 3D printer. [Download print files here](https://github.com/bigtreetech/Panda_Breath/tree/master/3D%20Model/Mounting%20solution).

    :::tip

    Make sure to select the mounting brackets that match your Panda Breath version. Otherwise, the brackets may not fit properly during installation.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install01.webp').default} alt="" width="35%"  />

### Installation Steps for Bambu Lab P1

1. Remove the top glass panel and place the printed riser on the top of the printer.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install01.webp').default} alt="" width="35%"  />

2. Locate and remove the mounting screw at the center of the right-side crossbar using a screwdriver.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install02.webp').default} alt="" width="35%"  />

3. Align the bottom mounting tabs of Panda Breath with the printer's alignment posts.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install03.webp').default} alt="" width="35%"  />

    :::tip

    If your Panda Breath is V1.1, remove the removable metal mounting bracket from the upper-left side of Panda Breath by unscrewing the two screws before installation. Otherwise, it may interfere with installation inside the P1 printer.

    :::

4. Fasten Panda Breath using the included M3x8 self-tapping screws.

    :::tip

    Tighten all screws securely, but avoid using excessive force. Over-tightening can strip the threads or crack the housing.

    :::

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install04.webp').default} alt="" width="35%"  />

5. Route and secure the power cable using the included cable clips, following the reference diagram.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install05.webp').default} alt="" width="35%"  />

6. Pass the cable through the rear opening in the printed riser.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install06.webp').default} alt="" width="35%"  />

7. Unplug the printer's original power cable, then connect the adapter cable as shown.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install07.webp').default} alt="" width="35%"  />

8. Remove the backing from the 2 included aluminum foil insulation pads and stick them to the bottom and right side of the left fan inside the printer to prevent the exterior from getting too hot during prolonged use.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install08.webp').default} alt="" width="35%"  />

Installation is now complete.

### Installation Steps for Bambu Lab P2S

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install01.webp').default} alt="" width="35%"  />

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p2s-install02.webp').default} alt="" width="35%"  />

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p2s-install03.webp').default} alt="" width="35%"  />

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p2s-install04.webp').default} alt="" width="35%"  />

4. Raise the printer bed to its highest position, then place Panda Breath inside the printer.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p2s-install05.webp').default} alt="" width="35%"  />

5. Align the upper-left mounting hole on the Panda Breath with the screw hole inside the printer, then secure it using the included self-tapping screw. Do not overtighten.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p2s-install06.webp').default} alt="" width="35%"  />

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/p-install07.webp').default} alt="" width="35%"  />

8. Peel off the backing from the two included aluminum-foil insulation pads, then attach them to the right side inside the printer where the airflow from Panda Breath directly blows. This helps prevent the Panda Breath housing from becoming too hot during long heating sessions.

### Installation Steps for Snapmaker U1

1. Take the two pre-printed clamp blocks, insert an M3x14 self-tapping screw into each hole on the left mounting bracket. Do not tighten the screws completely at this stage; leave them loose for adjustment.

    :::tip

    Because Panda Breath V1.0 and V1.1 have different thicknesses, make sure to download and print the mounting brackets that match your Panda Breath version.

    :::

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install03.webp').default} alt="" width="35%"  />

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install04.webp').default} alt="" width="35%"  />

3. Guide the Panda Breath cable toward the left side of the module. Thread the cable through the dedicated channel on the left mounting bracket. Gently pull the cable upward and out through the top exit of the channel.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install05.webp').default} alt="" width="35%"  />

4. Place the Panda Breath at the very back of the Snapmaker U1 chamber, gently pressing it against the transparent rear panel. Route the power cable out of the printer through the gap in the corner.

    :::tip[Important]

    Ensure the bottom of the Panda Breath is suspended—do not let it press on the cable.

    :::

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install06.webp').default} alt="" width="35%"  />

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install07.webp').default} alt="" width="35%"  />

5. Once the module is correctly positioned, fully tighten all four screws on the left mounting bracket to secure it in place.

    <ImageView src={require('@site/docs/panda-series/module/panda-breath/img/u1install08.webp').default} alt="" width="35%"  />

## Initial Connection

After the first power-on, the Panda Breath defaults to Auto Mode, and the green indicator on the button turns on. Note: Auto Mode requires the Panda Breath to be paired with a printer.

### Wi-Fi Connection Guide

- Connecting to the Panda Breath's Wi-Fi_AP Hotspot

    - Connect to the Panda Breath AP hotspot: `Panda_Breath_XXXXXXXXXX`. The default password is `987654321`.

- Accessing the AP Page

    - After connecting to the AP, open a browser on your computer and enter the default IP address: `192.168.254.1`. On mobile devices, the Panda_Breath Web UI will automatically appear, or you can manually enter the IP address in the browser: `192.168.254.1`. ## Introduction to Each Page

### Wi-Fi Page

<ImageView src={require('@site/docs/panda-series/module/panda-breath/img/wifi-ip.webp').default} alt="" width="35%"  />

1. Wi-Fi Page Selection: When you tap the icon, it will turn red to indicate that you are on the Wi-Fi page.

2. Current Wi-Fi Connection Status

3. IP Address Icon: Tap the icon to display a pop-up. If the IP field is blank, ensure Panda Breath shows "Connected." If it says, "Not Connected," check your network settings.

4. IP Address Display: Entering this IP in your browser will take you to the Web page, as long as you're on the same local network.

5. Wi-Fi Network Configuration Button: Tap to go to the Wi-Fi connection page

6. Host Name: The default hostname is "PandaBreath." If you forget the IP, you can use the hostname followed by "local" in the browser (e.g., http://PandaBreath.local). To modify the hostname, tap the text box to edit it, then tap "Set Hostname" to confirm.

7. Confirm Hostname Modification

### AP Page

<ImageView src={require('@site/docs/panda-series/module/panda-breath/img/wifi-ap.webp').default} alt="" width="35%"  />

1. AP Page Selection: When you tap the icon, it will turn red to indicate that you are on the AP page.

2. AP Hotspot Switch

3. AP Name: Default is `Panda_Breath_XXXXXXXXXX`

4. AP Password: Default is `987654321`

5. Hotspot IP: Default Hotspot IP is "192.168.254.1." If your router's IP range is "192.168.254.xxx," an IP conflict will prevent the device from binding to the printer. Change the Panda Breath hotspot IP to a different range.

6. Confirm Button: Tapping this button will restart the device. After the restart, you can connect to the AP hotspot.

 ### Control Page

- Features Overview
