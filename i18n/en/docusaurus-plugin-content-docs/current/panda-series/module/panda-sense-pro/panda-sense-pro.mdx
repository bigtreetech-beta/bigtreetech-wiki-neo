## Product Introduction

BIQU Panda Sense Pro is an 8-in-1 air quality monitoring module that tracks temperature, humidity, PM2.5, PM10, CO₂, formaldehyde, eTVOC, and AQI in real time, helping you better understand the air around your home, workspace, or 3D printing area.

## Technical Specifications

| Panda Sense Pro | Specification |
| --- | --- |
| Printed Housing Dimensions | 117.12mm x 79.78mm x 73.67mm |
| Power Input | USB-C DC5V 2A |
| MCU | ESP32-S3-WROOM-1-N16R8 |
| Current   | 0.52A (0.64A peak) |
| Logic Voltage | 3.3V |
| Additional Interfaces | I2C interface (compatible with Panda Sense module), reset button (for factory reset) |

</div>

## Product Features

- Monitor 8 key air metrics, including temperature, humidity, PM2.5, PM10, CO2, formaldehyde, TVOC, and AQI.

- With 3.5-inch color screen shows air data clearly at a glance.

- Physical button for quick page switching.

- This product carries a WIFI module, which can interact with information through the network

- Reset button for quick factory restore.

- Hardware flashing port for recovery if OTA update fails.

### Install the Screen and Mainboard

### Install the Printed Button

Insert the printed button cover through the button hole, with the notch facing the back of the housing so it aligns with the PCB button.

<ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/install2.webp').default} width="60%"/>

### Secure the Mainboard

Secure the mainboard to the printed housing with four M3x8 countersunk screws. When placing the mainboard, make sure the notch of the printed button cover aligns with the button on the mainboard, and that the side USB-C port aligns properly with the USB-C opening in the printed housing.

<ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/install3.webp').default} width="60%"/>

### Install the Screen Frame

Place the screen into the corresponding position, snap the printed screen frame into place, and secure it from both sides with four M3x8 countersunk screws.

<ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/install4.webp').default} width="60%"/>

### Install the Sensor

Insert the temperature and humidity sensor into the printed sensor holder, then secure it to the rear screw hole of the printed housing with one M3x5 socket head cap screw.

<ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/install5.webp').default} width="60%"/>

## Getting Started

### Initial Setup

1. Connect power: Use the included USB-C cable to power the device. The screen will turn on automatically.

2. Wait for warm-up: The eTVOC module requires a 5-minute warm-up. Please wait until warm-up is complete before checking eTVOC data.

3. Network setup: The device will display its hotspot name when no network is configured.

### Network Setup

When the network has not been configured, the display will show the hotspot name `"Panda_Sense_Pro_XXXXXXXXXXXX"`.

<ImageView src={require('./img/ap-name.webp').default} width="256"/>

Use a phone or computer to connect to this hotspot. The default password is `987654321`.

<ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/ap-select.webp').default} width="256"/>

After connecting to the hotspot, if the Web UI does not open automatically, manually enter `http://192.168.254.1/` in your browser.

Select a language.

<ImageView src={require('./img/select-lang.webp').default} width="256"/>

After scanning for Wi-Fi, select the Wi-Fi network you want to connect to, enter the Wi-Fi password, and tap Connect.

<ImageView src={require('./img/ap-connect.webp').default} width="256"/>

### Firmware Update

#### Feature Requests

If you would like to request new features for an upcoming Panda Sense Pro firmware release, please submit a feature request in the official [Panda Sense Pro GitHub repo](https://github.com/bigtreetech/PandaSensePro/issues).

#### OTA Update

Use a phone or computer browser to access the device Web UI.

- Connect to the device's AP. The default name is `Panda_Sense_Pro_XXXXXXXXXXXX`, Enter  `http://192.168.254.1` in your browser.

- If the device is already connected to a router, for example with the IP address 192.168.3.161, enter the IP address shown on the screen in your browser: `http://192.168.3.161`

Go to the Settings page.

- Tap to select the `.bin` file.

    <ImageView src={require('./img/select-bin.webp').default} width="256"/>

- Update successful.

    <ImageView src={require('./img/update-ok.webp').default} width="256"/>

#### flash_download_tool (Type-C Wired Flashing)

:::info[Note]

Panda Sense Pro normally supports OTA updates. Use this method only when OTA update fails or is unavailable.

:::

- If the CH340 driver is not installed on your computer, download and install the driver first: [CH341SER.EXE](https://www.wch-ic.com/downloads/CH341SER_EXE.html)

- Download the [Flash Download Tool](https://www.espressif.com/en/support/download/other-tools)

    <ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/download_esp_tool.webp').default} width="70%"/>

- Download the [Panda Sense Pro flashable firmware](https://github.com/bigtreetech/PandaSensePro/tree/master/recovery_tool/flash)

- Connect Panda Sense Pro to the computer via `Type-C`. A new COM port should appear in Device Manager.

    <ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/typec.webp').default} width="35%"/>

- Open `flash_download_tool_3.x.x.exe`, and configure the pop-up window as shown below:

    ChipType: `ESP32-S3`

    WorkMode: `Develop`

    LoadMode: `UART`

    <ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/download.webp').default} width="200"/>

- Configure the flashing software as shown below.

    <ImageView src={require('@site/docs/panda-series/module/panda-sense-pro/img/esp_tool_config.webp').default} width="400"/>

    1. Set the flashing address for each `.bin` file, and make sure the corresponding checkboxes are selected.

        - Write `panda_sense_pro_production.bin` to `0x0000`

    2. Set the COM port to the actual port. You can check it in your computer's `Device Manager` -> `Ports`. Set a suitable baud rate. `460800` is recommended.

    3. Click `START` to begin flashing. Wait until flashing is complete, then power-cycle the device.
