## Product Overview

**Panda Lungs** is an air purification module designed for Bambu Lab P1/X1 series printers. Key features include:

- **Print-Status Operation**: Automatically turns on when the printer starts printing and turns off when the print stops, completes, or fails.
- **Temperature and Humidity Display**: Displays real-time ambient temperature and humidity through TM1650 digital displays.
- **TVOC Monitoring**: Monitors and displays total volatile organic compounds (TVOC) concentration in real time.
- **Buzzer Alarm**: Automatically sounds an alarm when the TVOC level exceeds the threshold.
- **Web UI Remote Control**: Access the Web UI through a browser to view device status and configure parameters remotely.
- **Settings Memory**: All settings are automatically saved and retained after power-off.

### Control Panel Features

The Control Panel is the main configuration page and includes the following features:

#### Printer Paused — Auto-Off Delay

After the printer enters the **Paused** state, Panda Lungs keeps running and turns off automatically after the countdown ends.

- Range: **0 ~ 60 minutes**
- Default: **5 minutes**
- Operation: Tap `+` / `-` to adjust; press and hold to step quickly by $\pm$ 5 minutes
- When set to **0 minutes**, the fan turns off immediately after the printer is paused

#### Print Completed / Failed — Auto-Off Delay

After the printer enters the **Completed** or **Failed** state, Panda Lungs keeps running and turns off automatically after the countdown ends.

- Range: **0 ~ 60 minutes**
- Default: **5 minutes**
- Operation: Tap `+` / `-` to adjust; press and hold to step quickly by $\pm$ 5 minutes
- When set to **0 minutes**, Panda Lungs turns off immediately after the printer status changes.

#### Temperature Unit

Switch the temperature unit shown on the digital display:

| Option             | Description      |
| ------------------ | ---------------- |
| **Celsius / $^\circ\mathrm{C}$**    | Celsius, default |
| **Fahrenheit / $^\circ\mathrm{F}$** | Fahrenheit       |

#### TVOC Alarm Switch

| Status      | Description                                                                  |
| ----------- | ---------------------------------------------------------------------------- |
| On, default | When TVOC exceeds the threshold (0.6 mg/m³), the buzzer sounds automatically |
| Off         | Disables the buzzer alarm                                                    |

#### Save Settings

Tap the red **Save Settings** button at the bottom of the page to write all configurations to device storage. The settings will not be lost after power-off.

> **Important: You must Tap "Save Settings" after each parameter change for the settings to be saved permanently!**

#### Button Controls

The device panel is equipped with a multifunction button (IO9), which supports the following operations:

| Operation                                           | Function                                                         |
| --------------------------------------------------- | ---------------------------------------------------------------- |
| **Single press**                                    | Manually turn Panda Lungs on / off and enter manual control mode |
| **Triple press**                                    | Enter factory test mode after the device restarts                |
| **Press and hold (about 2 seconds)** (in test mode) | Exit factory test mode and return to normal mode                 |

#### Automatic Operation by Printer Status

Panda Lungs automatically turns on or off based on the printer's real-time status, according to the rules below:

| Printer Status           | Panda Lungs Behavior                                                         |
| ------------------------ | ---------------------------------------------------------------------------- |
| **Printing**             | Turns on immediately                                                         |
| **Paused**               | Stays on, then turns off automatically after the configurable countdown ends |
| **Print Completed**      | Stays on, then turns off automatically after the configurable countdown ends |
| **Print Failed**         | Stays on, then turns off automatically after the configurable countdown ends |
| **Idle / Not Connected** | Turns off immediately                                                        |

#### Manual Mode

- **Single press the button**: Manually turn Panda Lungs on / off and enter manual mode.
- In manual mode, Panda Lungs no longer follows the printer status automatically.

#### Digital Display Description

The device is equipped with 3 groups of TM1650 digital displays. Each group independently shows the following information:

#### TVOC Digital Display, Upper

| Display Content                     | Description                                                              |
| ----------------------------------- | ------------------------------------------------------------------------ |
| Number, such as `0.15`              | Sensor operating normally; displays real-time TVOC concentration (mg/m³) |
| Countdown number, such as `420`→`0` | Sensor warming up; wait until it is ready, up to 10 minutes              |
| `E.00`                              | Sensor response timed out. Please check the hardware connection          |

#### Temperature Digital Display, Middle

| Display Content        | Description                                                |
| ---------------------- | ---------------------------------------------------------- |
| Number, such as `25.6` | Real-time temperature value ($^\circ\mathrm{C}$ or $^\circ\mathrm{F}$, set in Control Panel) |
| `- -`                  | Sensor reading failed                                      |

#### Humidity Digital Display, Lower

| Display Content      | Description                       |
| -------------------- | --------------------------------- |
| Number, such as `48` | Real-time relative humidity (%RH) |
| `- -`                | Sensor reading failed             |

#### Buzzer Alarm Description

When **TVOC ≥ 0.6 mg/m³**, the buzzer automatically sounds continuously in the following pattern:

:::tip

Beep for 1 second → pause for 0.3 seconds → beep for 1 second → pause for 0.3 seconds → ... (loop)

:::

- The alarm can be turned off in Control Panel using the TVOC Alarm Switch
- After the alarm is turned off, this setting is retained after power-off and does not need to be set again each time

## FAQ

### Q1: The AP hotspot cannot be found after the device powers on.

**Solution:**

- Wait about 10 seconds. The device needs time to start up
- Make sure the device is powered properly through the USB-C port, 5V/1A or above
- Try powering the device off and on again

### Q2: Wi-Fi connection fails or keeps showing "Connecting".

**Solution:**

- Make sure the Wi-Fi password is entered correctly
- Make sure the router supports 2.4GHz Wi-Fi. Pure 5GHz networks are not supported
- If the connection fails multiple times, try restoring factory settings and configuring Wi-Fi again

### Q3: The printer cannot be scanned.

**Solution:**

- Make sure Panda Lungs and the printer are on the **same local network**
- Make sure the printer is powered on and connected to the network
- Try manually entering the printer IP address and SN to add the printer
- Check whether the printer Access Code is correct. You can find it in the printer settings.

### Q4: Settings are lost after the device restarts.

**Solution:**

- Make sure you Taped the **Save Settings** button after making changes

### Q5: The digital displays do not light up or show abnormal values.

**Possible Causes:**

- Sensor hardware connection issue
- Unstable I2C communication

**Solution:**

- Check the sensor connection
- Power the device off and on again

### Q6: How do I restore factory settings?

1. Go to the **Settings** page
2. Tap the **Factory Reset** button
3. Confirm the action. The device will automatically restart and clear all settings.

:::warning

Factory reset will clear all settings, including Wi-Fi, printer connection, and parameter settings. Please proceed with caution!

:::

### Q7: How do I access the Web UI using the hostname?

The default hostname is `PandaLungs`. Enter the following in your browser:

```
http://PandaLungs.local
```

If it cannot be accessed, you can view the device IP address on the **Wi-Fi & IP** page and access the Web UI directly using the IP address.
