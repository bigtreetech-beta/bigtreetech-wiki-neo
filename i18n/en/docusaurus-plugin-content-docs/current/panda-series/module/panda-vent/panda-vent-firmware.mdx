# Panda Vent Firmware Settings

## Initial Setup Guide {/* #first-connection */}

:::tip[tip]

Upon initial powering on or after a factory reset, you will see a blue flowing light effect, indicating that the Panda Vent is not yet bound to a printer.

:::

1. WiFi Page Selection: When you tap the icon, it will turn red to indicate that you are on the WiFi page.

7. Confirm hostname change.

### AP Page

<ImageView
    src={require('./img/panda-vent-step18.webp').default}
    width="45%"
/>

1. AP Page Selection: When you tap the icon, it will turn red to indicate that you are on the AP page.
2. AP Hotspot toggle (On/Off).
3. AP SSID (Hotspot Name). Default: `Panda-Vent-XXXXXXXXXX`.
4. AP Password. Default: `987654321`.
5. Hotspot IP (Default: `192.168.254.1`). If your router uses the same 192.168.254.xxx range, a conflict will occur, preventing the device from binding to the printer. Please manually change the Hotspot IP to a different subnet.
6. Tap Confirm and Restart. The device will reboot to apply the changes. Once the restart is complete, connect to the AP hotspot. See [How to connect to the AP hotspot](#ap-connect).

#### Light Mode {/* #light-mode */}

- **Simple Mode**
    - Best for users who want a single fixed lighting effect. The same effect is applied to all LEDs.

    - Available effects:

        | Effect                    | Description             |
        | ----------------------- | ---------------- |
        | Solid | Constant light with a fixed color |
        | Breathing | Light gradually fades in and out |
        | Flash | Light flashes quickly |
        | Flow | Light flows like a wave |
        | Marquee | LEDs light up in sequence and move |
        | Rainbow Cycle | Colors cycle automatically |
        | Multicolor | Rainbow gradient effect |

    - After selecting an effect, you can adjust color, brightness（$0\% \sim 100\%$）and speed（$0\% \sim 100\%$）。

- **Advance Mode / H2D**

    <ImageView
        src={require('./img/panda-vent-step28.webp').default}
        width="45%"
    />
    Advanced Mode Description:

    Configurable Printer States:
    - Configure different lighting effects for each printer operating state.
    - Tap the corresponding status button to separately set the effect, color, brightness, and speed for that state.

    | Status Button | Corresponding Printer Status | Default Color |
    |---------------|------------------------------|---------------|
    | Idle | Printer is idle | White (FFFFFF) |
    | Preparing | Downloading / slicing / preparing | Orange (F8A323) |
    | Printing | Printing in progress | Rainbow effect |
    | Paused | Printing paused | White (FFFFFF) |
    | Complete | Print completed | Green (00FF2A) |
    | Error | Printer error / alarm | Red (FF0000) |

- **Warning Hot Mode**

    <ImageView
        src={require('./img/panda-vent-step29.webp').default}
        width="45%"
    />
    Temperature Warning Mode Description:

    Divided into the following two levels:
    - **Safe State**: Green lighting effect, with static or flashing mode available
    - **Danger State**: Red lighting effect, with static or flashing mode available

    Brightness and speed can be set independently for each level.

- Vent Control (Button Operation)

    Panda Vent offers two vent operating modes, which you can switch using the **Illuminated Button** on the device panel:
    - **AUTO mode**: Default mode. Vent is automatically controlled based on printer status. Exhaust turns on during printing and turns off when the printer is idle. The illuminated button **remains off** in this mode.

    - **MANUAL mode**: The vent is fully controlled manually by the user. The illuminated button **blinks** at 0.5-second intervals.

    | Operation | Function |
    |-----------|----------|
    | **Single-click the Illuminated button** in AUTO mode | Switch to MANUAL mode and immediately toggle the vent state (open $\leftrightarrow$ closed) |
    | **Single-click the Illuminated button** in MANUAL mode | Toggle the vent state (on $\leftrightarrow$ off) |
    | **Long-press the Illuminated button for 6 seconds** | Switch from MANUAL mode back to AUTO mode |
    | **Long-press the BOOT button for 6 seconds** | Restore factory settings |

    :::tip[Tip]

   The mode switching status is saved automatically. After a power cycle, the device will restore the previous mode setting.

    :::
