## 简介

M4P是一款使用CM4 BTB头的4轴主板，可以完美地与CB1或CM4配合使用。

## 硬件规格

| Specification    | Manta M4P                          |
| ---------------- | ---------------------------------- |
| MCU              | ARM Cortex-M0+ STM32G0B1VET6 64MHz |
| Mainboard Power Supply | DC 24V                             |
| Driver Power Supply | Up to DC 48V (driver support required)         |
| Logic Voltage         | DC 3.3V                            |
| Maximum Current at Hot End     | 5A                                 |
| Maximum Current at Heated Bed Port | 10A                                |
| Maximum Current at Fan Port | 1A                                 |
| Total Fan Current       | 2.5A                               |
