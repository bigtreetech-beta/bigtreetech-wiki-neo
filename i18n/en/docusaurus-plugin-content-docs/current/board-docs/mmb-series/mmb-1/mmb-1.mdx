## Product Introduction

The BIGTREETECH MMB CAN V1.0 is a control board for multi-material 3D printing system, e.g., ERCF. It simplifies wiring by communicating via USB or CAN.

## Feature Highlights

- Equipped with onboard BOOT and RESET buttons, allowing users to enter DFU mode via USB to update firmware.
- Features a reserved I2C interface, which can be utilized for filament detection or other custom DIY functionalities.
- The power interface includes reverse polarity protection to prevent the board from burning out due to incorrect wiring during DIY setup.
- Supports both CAN and USB communication. The 120R CAN termination resistor is selectable via a jumper, and a CAN expansion interface is reserved.
- An ESD protection chip is integrated into the USB port to prevent the MCU from being damaged by electrostatic discharge.
- Utilizes an XT30 interface for CAN communication and power supply to simplify wiring.
- Stepper motor drivers support high and low voltage selection, providing flexibility for DIY applications.
