## Interface Introduction

### USB Power Supply

When the board is powered on, the power indicator light will turn on to confirm that the power supply is functioning correctly. The VUSB terminal on the board is the power selection terminal, which should only be shorted with a jumper when powering through USB.

	src={require('@site/docs/board-docs/mmb-series/mmb-2/img/mmb-can-v2-0-usb.webp').default}

### Servo Interface

:::info[Note]
This demonstration shows using the MMB's onboard DC-DC module to power the servo.

The MMB CAN V2 features an added 3A resettable fuse at the servo output port to prevent the power supply from triggering short-circuit protection, making it fully capable of handling this application. However, please ensure it is connected to the MOT1 or MOT2 connector, minimize simultaneous movements, or enable `servo_always_active: 1` to maintain continuous control signals even after a servo reset.
:::
