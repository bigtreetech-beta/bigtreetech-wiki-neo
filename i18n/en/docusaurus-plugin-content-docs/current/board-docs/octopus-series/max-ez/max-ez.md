## Product Overview

The Octopus MAX EZ is a 32-bit printer mainboard optimized and upgraded from the Octopus Pro. It features a proprietary stepper motor driver board that enhances safety and user experience, and adds a range of features not available on the Octopus Pro.

## Specification

| Specification    | Octopus Max EZ                     |
| ---------------- | ---------------------------------- |
| MCU              | Arm Cortex-M7 STM32H723ZET6 550MHz |
| Mainboard Power Supply | DC 12V - DC 24V                    |
| Driver Power Supply | Up to DC 56V (driver support required)          |
| Heated Bed Interface Power Supply | DC 12V - DC 24V                    |  |
| Maximum Heated Bed Current     | 10A (12A peak)                     |
| Maximum Heated End Current | 5.5A (6A peak)                     |
