## Product Overview

The BIGTREETECH CB2 is an alternative solution designed to address the shortage of Raspberry Pi CM4 units. All output signals utilize two 100-pin micro BTB high-speed connectors for quick and easy connection to external expansion boards (including Gigabit Ethernet, HDMI, etc.). It features onboard 2.4GHz and 5GHz Wi-Fi and Bluetooth 5.2.

## Product Features

- CPU: Rockchip RK3566, Quad-core Cortex-A55 @ 1.8GHz
- GPU: Mali-G52 1-Core-2EE
- NPU: 0.8 TOPS NPU
- RAM: 2GB/4GB LPDDR4
- On-board eMMC
- MIPI DSI display support
- Camera: 2-Lane MIPI CSI2
- 3 USB 2.0 ports, 1 USB 3.0 port
- PCIe 2.1 1x1 Lane
- Gigabit Ethernet + 433Mbps Wi-Fi + Bluetooth 5.0
- 40-pin GPIO
- BTB socket identical to the Raspberry Pi CM4
