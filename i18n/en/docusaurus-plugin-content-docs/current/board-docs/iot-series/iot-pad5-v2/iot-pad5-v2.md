## Product Overview

The BIGTREETECH Pad5 V2.0 is an expansion display board compatible with CM-series modules, featuring special enhancements for the Raspberry Pi CM5 for easier operation.

## Product Features

1. Compatible with multiple SoC compute modules, including CM5, CM4, CB2, and CB1.

2. The onboard USB OTG port allows convenient image flashing for compute modules with eMMC storage.

3. Supports USB 3.0 communication when used with the CM5.

4. Includes a reserved XH2.54 2-pin CAN interface for DIY applications.

5. Equipped with a 5-inch IPS capacitive touchscreen for improved display quality and a better user experience.

6. All interfaces feature ESD protection for enhanced product stability.

7. Supports one-button power on/off when used with the CM5, making operation more convenient.

8. Features an onboard host-side USB hub: one channel connects to the UART port via CH340E, while another connects directly to USB OTG, making image flashing and SoC debugging more convenient.

9. Offers multiple USB interface options for greater flexibility in DIY applications.
