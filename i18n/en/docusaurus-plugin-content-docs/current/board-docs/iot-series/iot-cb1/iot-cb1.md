## Product Overview

The BIGTREETECH CB1 is an alternative solution designed to address the shortage of Raspberry Pi CM4 units. All output signals are connected via two 100-pin micro BTB high-speed connectors to an external expansion board for quick and easy connectivity (including 100Mbps Ethernet, HDMI, and more). It features onboard 2.4GHz Wi-Fi.

## Key Features

- CPU: Allwinner H616, quad-core Cortex-A53 @ 1.5GHz
- GPU: Mali G31 MP2, supports OpenGL 3.2
- RAM: 512MB/1GB DDR3L SDRAM
- Display: HDMI 2.0a port, supports 4K displays
- USB 2.0 ports
- 100Mbps Ethernet + 100Mbps Wi-Fi
- BTB socket identical to the Raspberry Pi CM4
