## Product Introduction

BIGTREETECH EBB36 CAN V1.0/V1.1/V1.2 is a toolhead CAN board designed to simplify toolhead wiring. The EBB 36 can communicate via USB port or CAN bus, greatly simplifying 3D printer toolhead wiring and routing.
