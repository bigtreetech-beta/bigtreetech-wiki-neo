## Introduction

EBB 2209 (RP2040) is a toolhead board adapted for the Voron Stealthburner. It connects via a CAN line to simplify toolhead wiring.

## Hardware Specifications

| Specification                 | EBB 2209 RP2040                 |
| ----------------------------- | ------------------------------- |
| MCU                           | RP 2040                         |
| Extruder Driver               | TMC 2209                        |
| Temperature Sensor (TH0)      | NTC / PT1000                    |
| Temperature Sensor (MAX31865) | PT100 / PT1000 2 wire or 4 wire |
| Hotend Maximum Current        | 5A                              |
| Fan 2Pin                      | x2                              |
| Fan 4Pin                      | x1                              |
