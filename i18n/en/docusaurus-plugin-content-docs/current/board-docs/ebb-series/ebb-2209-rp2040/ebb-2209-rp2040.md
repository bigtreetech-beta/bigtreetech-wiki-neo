## 简介

EBB 2209 (RP2040) 是一个适配 voron stealthburner 的工具头板。通过 CAN 线进行连接以简化工具头接线

## 硬件规格

| Specification         | EBB 2209 RP2040                 |
| --------------------- | ------------------------------- |
| MCU                   | RP 2040                         |
| 挤出机驱动            | TMC 2209                        |
| 温度传感器 (TH0)      | NTC / PT1000                    |
| 温度传感器 (MAX31865) | PT100 / PT1000 2 wire or 4 wire |
| 热端最大电流          | 5A                              |
| 风扇 2Pin             | x2                              |
| 风扇 4Pin             | x1                              |
