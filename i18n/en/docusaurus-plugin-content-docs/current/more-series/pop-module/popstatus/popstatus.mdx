## Product Introduction

**PopStatus —— Print. Pulse. Perform.**

    Designed exclusively for the Snapmaker U1, PopStatus is a smart status light bar that delivers real-time visual feedback on your printer's activity while adding a stylish touch to its look. Thanks to its diffused lighting design, PopStatus produces smooth, even illumination, so you can instantly recognize printing, idle, completed, and error states at a glance.

- **Fully Customizable Status Lighting**

    Via the Web UI or BIQU App, you have full freedom to adjust the color, brightness, and dynamic mode (such as breathing, flowing, or flashing) for each printer state. This lets you give every status its own unique visual signature.

- **Rhythm-Light Sync**

    A built-in high-sensitivity microphone enables the lighting to react instantly to ambient sound. Your PopStatus will pulse and flow in time with the noises in your workspace — adding a genuine 'heartbeat' to your setup.

- **Premium Finish**

    The housing is crafted from aluminum alloy with an anodized finish, delivering both a high-end feel and long-lasting durability. The front panel features a laser-engraved acrylic cover, which you can swap out for a self-printed lithophane cover, offering even more room for personalization.

:::tip

Note: You'll find spare end caps (with and without cable openings) in the box. By default, the connector on the light board sits on the left, but you can flip the internal light board to route the cable from the right side. To do this, remove both end caps, reverse the  light board and reconnect the cable. Attach the closed end cap on the left and the opening end cap on the right. **Keep in mind**: Flipping the light board will also reverse the direction of the lighting effects. To fix this, enable the reverse option in the Web UI (see the firmware section for details).

:::

- After raising the heated bed, peel off the release liner from the adhesive backing of PopStatus, align the side of PopStatus with the heated bed, then press it firmly into place to ensure a secure bond. **Recommendation**: For best alignment, install the build plate *before* attaching PopStatus. If your build plate has a 3D-printed handle, shift PopStatus slightly downward to avoid any interference.

    <ImageView src={require('@site/docs/more-series/pop-module/popstatus/img/popstatus-v1-fix-controller.webp').default} alt="" width="45%" />

    <ImageView src={require('@site/docs/more-series/pop-module/popstatus/img/popstatus-v1-cable-routing-3.webp').default} alt="" width="45%" />

- Plug the communication cable into the control board. Then take the Type-C connector on the shorter side of the power cable (near the coiled section) and insert it into the rear power port. Use the included tape to secure it as shown, making sure the power cable exits from the back of the heated bed.

    <ImageView src={require('@site/docs/more-series/pop-module/popstatus/img/popstatus-v1-cable-routing-2.webp').default} alt="" width="45%" />

Run the power cable along the gap between the left side of the toolhead and the machine frame. Secure it with the provided tape at the location shown in the image, then arrange the cable so it exits toward the rear of the machine. Finally, connect it to a 5V power supply. **Important**: Do not plug the power cable into the USB port on the back of the machine. That port is meant for data communication only and cannot supply enough power. Connecting PopStatus to it may cause insufficient power and prevent it from working properly.

### Wi-Fi Connection Guide

- Connect to the Device Wi-Fi Hotspot

    - Connect to the AP hotspot generated by the device: `pop_Status_XXXXXXXXXX`. The default password is `987654321`.

    - Once connected, PC users can enter the default IP in a browser: `192.168.254.1`. Mobile users should see the Web UI pop up automatically; otherwise, enter `192.168.254.1` in a mobile browser. (Note: This is the factory default IP).

### Binding the Printer

#### Method 1: Manually Bind Printer

<ImageView src={require('./img/popstatus-v1-manual-binding-en.webp').default} width="45%" />

    1. In the Web UI, navigate to the Printer page. The icon will be highlighted in red.

    2. Tap Scan to look for printers on the same local network.

    3. Leave the default port number as is. No changes are required.

    4. Select any printer that  appears in the list.

    5. Enter the IP address of the printer you want to bind. (To view the printer IP address: on the printer screen home page, tap Settings-Wifi in sequence. The IP address can be seen below the connected Wifi name).

        <ImageView src={require('./img/popstatus-v1-view-machine-ip-en.webp').default} alt="" width="45%" />

    6. Keep it the same as the default port number.

    7. Tap Bind.

## User Interface Description

### Wi-Fi Interface

<ImageView src={require('./img/popstatus-v1-wifi-interface-description-en.webp').default} width="45%" />

    1. WiFi Page Selection: When you tap the icon, it will turn red to indicate that you are on the WiFi page.

    2. Current Wi-Fi connection status.

    3. IP address troubleshooting prompt. If the IP text box is empty, ensure the status shows "Connected." If "Not Connected," please reconfigure network settings.

    4. IP Address display (used to access the Web UI from other devices on the same network).

    5. Network configuration button. Tap to return to the "Wi-Fi Connection Interface".

    6. Hostname display. You can access the Web UI via the hostname. Default: `PopStatus`. If you forget the IP, enter the hostname followed by .local (e.g., `http://PopStatus.local`). Tap the text box to edit and "Set Hostname" to confirm.

    7. Confirm hostname change.

### AP Interface

<ImageView src={require('./img/popstatus-v1-ap-interface-description-en.webp').default} width="45%" />

    1. AP Page Selection: When you tap the icon, it will turn red to indicate that you are on the AP page.

    2. AP Hotspot toggle (On/Off).

    3. AP SSID (Hotspot Name). Default: `Pop_Status_XXXXXXXXXX`.

    4. AP Password. Default: `987654321`.

    5. Hotspot IP (Default: 192.168.254.1). If your router uses the same 192.168.254.xxx range, a conflict will occur, preventing the device from binding to the printer. Please manually change the Hotspot IP to a different subnet.

    6. Confirm and Restart. The device will reboot to apply changes.

    3. Current Firmware version.

#### Lighting Modes

    - **Rhythm-Light Sync**

        After selecting this effect, you can adjust the brightness from 0% to 100%. The lighting response changes according to the surrounding audio level.

    - **Advanced Mode**

        <ImageView src={require('./img/popstatus-v1-control-panel-description-2-en.webp').default} width="45%" />

            - Set up different lighting effects for each printer operating status..

            - Tap any status button to independently configure its mode, color, brightness, and speed.

                | Status | Corresponding Printer Status | Default Color |
                |----------|--------------|----------|
                | Idle | Printer idle | White (FFFFFF) |
                | Preparing | Downloading/slicing preparation in progress | Orange (F8A323) |
                | Printing | Printing in progress | Rainbow effect |
                | Paused | Print paused | White (FFFFFF) |
                | Completed | Print completed | Green (00FF2A) |
                | Error | Print error/alarm | Red (FF0000) |
