# CryoGrip Pro 系列冷打板

<div class="div-table">

<ImageView src={require('./img/product.webp').default} width="50%" class="right-image"/>

## 产品简介

**CryoGrip Pro** 系列打印板是 BIQU 专为解决 3D 打印因为粘附力不足而导致的打印失败, 热床反复加热导致电费增加, 热床寿命降低而设计的新一代弹簧打印面板。仅需要少量的加热就可以提供超强的粘附力和打印稳定性，用户可以实现低温下稳定的打印。

**CryoGrip Pro** 系列包含 **Frostbite (霜冻)** 系列和 **Glacier (冰川)** 系列。

## CryoGrip Pro Glacier | 冰川系列

**CryoGrip Pro Glacier** 表面纹理细腻。它属于对新手十分友好的打印板类型，用户可以在不涂胶水的情况下打印大部分的常见耗材。仅在 45°C 低热床温度下打印 PLA 时就能够牢牢固定模型，PETG 则只需 60°C；这将有效减少翘曲和“炒面”，显著降低大尺寸模型的失败风险。此外，该涂层能显著节能，降低堵头风险，更加环保。

- **省电环保**: 使用极低的热床温度即可确保牢固粘附。大幅度节约热床加热所需的能量，降低您的电费账单，对地球环境更加友好。

    <ImageView src={require('./img/g1.webp').default} width="30%"/>

- **材料适配广泛**: 支持 PLA, PETG, ABS, ASA, PA, PC, 尼龙, 玻纤等大多数打印耗材。无需担心适配问题，您可以放心地使用 Glacier 打印需要的工程耗材。

    <ImageView src={require('./img/g2.webp').default} width="30%"/>

- **可靠粘附，告别炒面**: 在确保面板无脏污的情况下，粘附力相较于 PEI 钢板有着数倍的提升。大型模型也不用担心脱落风险。

    <ImageView src={require('./img/g3.webp').default} width="30%"/>

- **底面质感**: 底面效果相较于 PEI 钢板更加细腻。为您提供更佳的成品感官体验，增添设计的整体统一感。

    <ImageView src={require('./img/g4.webp').default} width="30%"/>

## CryoGrip Pro Frostbite | 霜冻系列

**CryoGrip Pro Frostbite** 表面纹理相较于 Glacier 更粗糙。特殊的处理工艺让其在极低温度下拥有超前的表现。仅在 30°C 低热床温度下打印 PLA 时就能够牢牢固定模型，PETG 则只需 50°C；ABS 请按照默认温度设置。Frostbite 专为 PLA/PETG/ABS 设计，不支持其他耗材。

- **省电环保**: 仅需极低热床温度即可确保牢固粘附。大幅度节约热床加热所需能量，超级省电环保。

    <ImageView src={require('./img/f1.webp').default} width="30%"/>

- **超强粘附，放心开打**: 即使在 30°C 的环境下也能紧紧锁住模型。大幅度提升打印成功率，尤其是在打印大面积模型时，您不用担心失败。

    <ImageView src={require('./img/f4.webp').default} width="30%"/>

- **适配 80% 常用耗材**: 专为 PLA/PETG/ABS 设计。能够精准满足绝大多数用户的核心需求，无需为冗余的性能支付额外的成本。

    <ImageView src={require('./img/f2.webp').default} width="30%"/>

- **底面质感**: 底面效果比 Glacier 更加粗糙且富有层次感。能为特定设计件增添独特的视觉纹理。

    <ImageView src={require('./img/f3.webp').default} width="30%"/>

## 拓竹机型二维码识别

目前仅有X1C, P2S, H2系列机型支持打印过程中的二维码识别，你可以在打印选项中关闭打印板检测功能以实现顺利的打印。

您也可以在社区论坛评论中发布需求适配的信息，我们的平台会收集相关信息，根据你的反馈更快的适配你想要的机型。

- **X1C**:

    - 目前的Glacier打印面板会在售出时随打印面板赠送X1C可以使用的二维码，无需担心识别失败的问题；

    - 目前的Frostbite已适配了拓竹低温打印板的二维码，正确的设置下，无需担心识别失败的问题；

- **P2S**: 目前的Glacier和Frostbite不支持打开打印板检测功能下实现识别，我们正在努力适配中，你可以在开源社区打印适配的模型自行适配；

- **H2系列**：目前的Glacier和Frostbite不支持打开打印板检测功能下实现识别，我们正在努力适配中，你可以在开源社区打印适配的模型自行适配；

### CryoGrip Pro 打印质量与预期不符

CryoGrip Pro设计目标时为了用户在打印时可以更加安心省电，但有时结果会不尽人意，以下是排除附着力问题的几个建议。

- **清洁打印表面**: 我们建议按照规定的方法不时地清洁打印面板，以确保打印表面没有任何打印碎屑, 残留胶水或手部油脂残留。你手上的油脂沾到打印表面也会影响打印面的附着力。`Frostbite`可以使用`肥皂水`；`Glacier`可以使用`酒精`或`肥皂水`；

- **检查设置的热床温度和使用的耗材**: 大多数打印机都会在切片软件中正确的设置打印温度，如果修改了打印配置，建议在打印失败后检查热床的设置温度。有时，你也可以根据耗材来设置热床温度。

- **按照耗材需求的参数修改打印设置**: 不同的耗材因为特性不同，错误的冷却和保持仓温都会影响CryoGrip的粘附力。

    - 对于打印PLA耗材，如果出现粘附力问题，可以适当降低仓温保持，高速打印的情况下，可以降低辅助冷却风扇的转速。

    - 对于打印PETG耗材，如果出现粘附力问题，可以通过保持仓温，降低打印速度再次尝试。

- **在切片软件中调整首层温度**: 大多数打印机都会在切片软件中正确的设置首层打印温度，你可以通过切片软件的设置，修改首层的相关设置，以获得更好的粘附力效果。例如：调整首层的层高和基础宽度，首层的热床温度和喷嘴温度，打印速度等，这些设置将为打印的其余部分提供一个良好的基础，并将增加打印成功率。

- **打印件过于黏附，难以取下**:

    - 等待打印面板温度降至室温水平；

    - 弯曲弹簧打印面板待模型脱模后，借助铲刀轻轻撬起模型；

    - 后续的打印过程中，根据材料特性适当降低温度，减少粘附力。

- **打印板表面被铲刀划伤**: 为实现低温免涂胶增稳打印，增稳低温打印板的表面涂层硬度相对低于纹理 PEI 板，因此更容易被铲刀划伤。不过，正常使用中出现的轻微划痕不会显著影响打印粘附性能。

- **打印件在打印板上留下痕迹**: 部分残留痕迹可通过酒精擦拭或洗洁精清洗去除。 少量无法完全去除的色斑或首层痕迹不影响打印板的长期使用与性能。
