## PyroGrip | 高温打印板系列

**PyroGrip** 是通过在不锈弹簧钢板上粉末喷涂一层 PEI 粉末而成。纹理 PEI 打印板最大的特点是它的表面具有特殊的粗糙纹理，并能给打印件底面带来同样的纹理。纹理PEI打印板适用的材料非常广泛，在绝大部分情况下不需要使用粘合剂，便能表现出非常好的粘附力，所以使用非常方便。纹理 PEI 打印板上面的 PEI 涂层非常坚固，所以寿命也很长。

**纹理PEI与光面PEI的区别**: 光面 PEI 打印板（Smooth PEI Plate）是将 PEI 膜通过高耐热的 3M 胶贴合在弹簧钢板上得到的。光面 PEI 打印板能够给打印件带来平整的底面，适用于对底面有平整度要求的场景。PyroGrip在适配`拓竹 A1 mini` `拓竹 A1` `拓竹 P1P` `拓竹 P1S` `拓竹 X1C` `拓竹 X1E`机型时，可以选择一面光面PEI和一面纹理PEI的型号。

- **热粘冷脱，经久耐用**: PyroGrip打印面板可以在所需温度下提供强大的粘附力，而在冷却后实现轻松的取模。

## 故障排除

### PyroGrip打印质量与预期不符。

- **清洁打印表面**: 我们使用清水搭配洗洁精不时地清洁打印面板，以确保打印表面没有任何打印碎屑, 残留胶水或手部油脂残留。你手上的油脂沾到打印表面也会影响打印面的附着力。

- **正常的涂层磨损**: 擦嘴过程用于在开始打印任务前清洁喷嘴。每次启动打印任务后，清洁动作会自动执行，无需人工干预。清洁动作包括粗擦和精擦，粗擦主要用于清洁热嘴上的残余废料，精擦则用于清洁热嘴端面保证光洁。精擦动作指的是喷嘴在构建板的特定位置向下蹭动1~2毫米。因此，如果您发现打印板的这个区域有一些涂层磨损，不用担心，这是正常现象。

- **略微调整热床温度**:

    - 热床温度会极大地影响材料与PyroGrip纹理PEI打印板之间的作用力，热床温度越高，则粘附力越强。如果你觉得粘附力不够，可以适当提高床温以增加粘附力。

    - 部分耗材在热床温度过高时出现堵头或明显象脚现象，此时可以适当降低热床温度。

- **合理地使用裙边和支撑**: 一些细而高的物体在高速打印时更容易被喷嘴刮倒。建议在打印这些物体时使用合理的支撑以优化整体的受力结构，从而降低打印件脱落的风险。

- **再次检查热床温度和使用的耗材类型**: 大多数耗材类型在切片软件中都有默认预设，但如果您自定义了配置文件，则建议再次检查热床设置的温度。有时，您还需要根据所使用的耗材类型及其特性来调整温度。

- **在切片软件中调整首层设置**: 为增强附着力，您还可以调整首层的切片设置。 切片软件中包含的切片配置文件已经预先配置了最佳测试值。 但如果出现问题，可以尝试建议从0.5mm线宽和0.25mm层高开始打印以获得最佳效果。 这些设置将为模型提供良好基础，提高成功率。
