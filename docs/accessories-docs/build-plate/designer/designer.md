## Designer 系列

:::warning[使用寿命]

转印膜是消耗品。为延长转印纹理的使用寿命,请时不时变换打印对象在打印板上的打印位置。请小心使用刮刀,以免损坏贴膜表面。

:::

**Designer** 利用激光技术在薄膜表面形成精细的光学纹理并通过耐高温胶附着在特制的基板上，该薄膜具有增透, 反射和偏振等光学特性。在打印过程中，高温的聚合物进入这些纹理，将其转印到打印模型的底面。 不同的光照角度下，模型底面呈现出不同的色泽效果。

- **高温粘附力高，低温取模轻松**: 高温下提供优秀的粘附力，可以放心的开始打印；打印完成冷却后，只需轻轻弯曲面板即可轻松取模。

## 拓竹机型二维码识别

目前仅有X1C, P2S, H2系列机型支持打印过程中的二维码识别，你可以在打印选项中关闭打印板检测功能以实现顺利的打印。

您也可以在社区论坛评论中发布需求适配的信息，我们的平台会收集相关信息，根据你的反馈更快的适配你想要的机型。

- **X1C**: 目前的Designer打印面板相应位置有适配X1C的打印板二维码，无需担心识别失败的问题；

- **P2S**: 目前的Designer打印面板不支持打开打印板检测功能下实现识别，我们正在努力适配中，你可以在开源社区打印适配的模型自行适配；

## 故障排除

- **清洁打印表面**: 我们使用清水搭配洗洁精不时地清洁打印面板，以确保打印表面没有任何打印碎屑, 残留胶水或手部油脂残留。你手上的油脂沾到打印表面也会影响打印面的附着力。

- **打印设置建议**: 使用Designer系列打印板打印模型时,请将打印板类型设置为光面 PEI/高温板,并遵循热床温度设置建议。

- **激光雷达校准失败**: 由于转印膜的反射效果,部分带有激光雷达的打印机将无法正常工作。建议跳过激光雷达校准程序,或使用普通打印板进行校准。
