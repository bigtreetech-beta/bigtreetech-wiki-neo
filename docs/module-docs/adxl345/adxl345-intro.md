## 简介

ADXL345 加速度计模块是针对打印机共振补偿设计的模块，使用 USB 连接简化接线。

## 硬件规格

|ADXL345|Specification|
|-|-|
|MCU|RP2040 Dual ARM Cortex-M0+ @ 133MHz|
|尺寸|$33.25mm \times 15.5mm$|
|传感器|ADXL345|
|传感器连接方式|4line SPI|
