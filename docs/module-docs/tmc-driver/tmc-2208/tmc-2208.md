## 产品简介

TMC2208是一款适用于两相步进电机的超静音电机驱动器IC。其引脚布局与众多传统驱动器兼容。TRINAMIC先进的stealthChop2斩波技术确保无噪声运行、最高效率及最佳电机扭矩。其快速电流调节功能配合可选的spreadCycle组合，可实现高度动态的运动控制。集成功率MOSFET可处理高达1.4A RMS的电机电流。保护与诊断功能保障运行稳健可靠。简易的UART接口拓展更多调校与控制选项，特定应用调校参数可存储于OTP存储器。
