# Panda Treat 操作

:::info[Note]

本节假定你已经按照硬件部分提供的说明，将 Panda Treat 组装并安装到打印机上。

:::

## 首次设置

安装 Panda Treat 并打开打印机后，Panda Treat 会显示用于首次设置的 QR code。

<ImageView src={require('./img/factory-reset-lcd-pixel-circle.webp').default} width="120"/>

使用手机或电脑扫描 QR code，连接到 Panda Treat 在设置过程中创建的 WiFi hotspot。扫描 QR code 后，可能需要点击弹窗才能让设备执行该操作。连接到 Panda Treat hotspot 最多可能需要 30 秒，并且你可能需要打开设备上的浏览器，以提示设备自动跳转到 Panda Treat 设置页面。

设置过程中，浏览器可能会显示 `No internet connection`；这是正常现象，因为设备是直接连接到 Panda Treat access point。设置页面加载后，应如下面图片所示。如果扫描 QR code 后 60 秒内仍未加载，你可以直接在浏览器中输入 `192.168.254.1` 并按 Enter 访问配置页面。

<ImageView src={require('./img/captive-portal.webp').default} width="256"/>

选择首选语言并按 `Next`。在 WiFi 页面，点击 `Scan`，扫描完成后选择与打印机相同的本地 WiFi 网络，输入 WiFi 密码，然后按 `Connect`。

<ImageView src={require('./img/wifi-selection.webp').default} width="256"/>

Panda Treat 连接到所选网络时，请保持浏览器页面打开。如果你误关页面，或与 Panda Treat hotspot 断开连接，可以再次扫描屏幕上的 QR code 回到 WiFi 设置页面。连接成功后，按 `Binding` 继续进入打印机绑定页面；如果需要停留在 WiFi & IP 页面，则按 `Cancel`。

<ImageView src={require('./img/wifi-connected-binding.webp').default} width="50%"/>

在 `Bind to Printer` 页面，按 `Scan` 搜索可用打印机。扫描最多可能需要 60 秒。列表中出现打印机后，选择已安装 Panda Treat 的那台打印机。如果找不到打印机，请检查 Panda Treat 是否与打印机处于同一 WiFi 网络，并确认打印机已开机。不建议使用访客网络，因为它们通常会阻止设备之间通信。

如果以上条件都满足但仍找不到打印机，则需要手动输入详细信息。按照下图所示，在打印机设置页面中查找并输入打印机序列号 (SN)，access code 和 printer IP。

<ImageView src={require('./img/find-access-code.webp').default} width="70%"/>

<ImageView src={require('./img/binding-to-printer.webp').default} width="256"/>

确认打印机信息正确后，按 `Bind`。当出现 `Binding successful` 时，Panda Treat 已绑定到指定打印机。随后，Panda Treat 显示屏会根据打印机状态显示动画。

<ImageView src={require('./img/bound-to-printer.webp').default} width="256"/>

WiFi 和打印机绑定完成后，在同一网络中的设备上使用 `pandatreat.local` 进行正常的 Panda Treat 操作。

## 软件更新

Panda Treat 会不定期收到软件更新，这些更新会为产品增加常规改进，并可能启用额外功能。强烈建议定期检查软件更新，尤其是在首次设置完成后立即检查。

检查软件更新最简单的方法是让 Panda Treat 自动检查。只要它连接到具有互联网连接的 WiFi 网络，就会自动执行检查。如果有可用更新，显示屏会显示如下图所示的图像。使用与 Panda Treat 位于同一 WiFi 网络的设备扫描 QR code，它会带你进入可安装更新的页面。

<ImageView src={require('./img/update-rounded.webp').default} width="120"/>

你也可以点击右上角 hamburger menu 中的设置选项，手动访问该页面。

<ImageView src={require('./img/access-settings-menu.webp').default} width="70%"/>

页面加载后，应出现一个弹窗，提示有可用更新。选择 `Confirm` 继续更新。更新过程中，Panda Treat 应保持连接互联网，并且不要断开电源。更新通常分两个阶段进行，进度会通过 Web UI 以及 Panda Treat 显示屏显示。

<ImageView src={require('./img/update-available-popup.webp').default} width="360"/>

如果 Panda Treat 未连接互联网，仍可手动下载并安装更新文件。要手动应用更新，请先从 [CMYLabs/PandaTreat releases](https://github.com/CMYLabs/PandaTreat/releases) 下载最新软件。

之后，进入设置菜单并点击 `Manual update`。对于 firmware update，请选择文件名中包含 `firmware` 的文件。对于 UI update，请选择文件名中包含 `UI` 的文件。

<ImageView src={require('./img/manual-update.webp').default} width="70%"/>

更新安装完成后，Panda Treat 会重启并重新连接到 WiFi 网络。

## 设置

Panda Treat 提供多种设置，可用于调整打印质量和其他重要功能。

点击右上角 hamburger menu 中的设置选项，然后进入所显示页面右下角的设置标签，即可访问设置菜单。

<ImageView src={require('./img/access-settings-menu.webp').default} width="70%"/>

软件版本和更新已在软件更新部分说明。其他可用设置包括：

- `Ink Strip Spacing`: 调整打印走线之间使用的间距。它可以增加或减少每条打印线之间的间距。如果清洁墨盒后仍发现打印线之间有重叠或间隙，此设置会很有用。故障排除部分包含更多关于如何使用此设置的信息。
- `Set Emergency Stop`: 除非特定诊断流程另有说明，否则应保持启用。
- `Factory Reset`: 将 Panda Treat 恢复为出厂设置。

<ImageView src={require('./img/settings-page.webp').default} width="70%"/>

## 在物品上打印

### 所需内容

使用 Panda Treat 时，你会打印到具有特定宽度，长度和高度的物品上。你需要告诉 Panda Treat 这些测量值。

要测量物体的宽度，长度和/或直径，可以使用简单的尺子，也可以使用游标卡尺等更复杂但更准确的工具。你也可以使用下方链接打印自己的游标卡尺。

测量高度时，建议使用下方所示的可打印工具。该工具的摆臂可以扫过你计划打印的物体，从而检查表面一致性，并确保没有可能刮碰的高点。

<ImageView src={require('./img/measure-object.webp').default} width="50%"/>

- Printable height measurement tool with instructions - Coming soon
- Printable vernier callipers: [vernier-caliper-ruler-two-lengths](https://makerworld.com/en/models/198362-vernier-caliper-ruler-two-lengths)

### 非自定义形状准备工作流

现在你已经到了可以在物体上打印的阶段。以下步骤将引导你完成非自定义形状的打印流程。

首先，确保打印表面已经达到食品安全要求。方法是在打印机成型板范围内铺上一片硅胶烘焙垫或烘焙纸。

<ImageView src={require('./img/prepared-surface.webp').default} width="45%"/>

接下来，访问 Panda Treat 的打印界面。你可以使用设备扫描 Panda Treat 显示屏上的 QR code。执行此操作时，请始终确保设备与 Panda Treat 位于同一 WiFi 网络。

:::info[Note]

扫描代码后，建议将页面加入书签并添加到主屏幕作为快捷方式，这样以后需要时可以快速访问。

:::

<ImageView src={require('./img/qr-to-print-page.webp').default} width="70%"/>

在首页选择你想要打印的物品类型。`Cookie / Cake` 用于固体物品，`Beverage` 用于饮品或泡沫等表面。

<ImageView src={require('./img/item-type-selection.webp').default} width="70%"/>

选择最接近该物品的形状 (A)。对于简单的饼干，蛋糕或饮品表面，请使用圆形或方形选项。自定义形状工作流将在本指南的另一部分说明。

输入你要打印的物体尺寸 (B)。即使你只会打印到物体的一小部分区域，也建议测量整个物体的尺寸，然后使用 zoom slider (E) 缩放图像，使其适配你计划打印的较小区域。如果输入的尺寸小于实际物体，后续打印定位标记时，可能会难以将物体放入标记范围内。

:::info[Height]

高度是最重要的数值，因为高度不正确可能导致打印头拖过物体。请仔细测量物品，如果表面不平整，请增加少量安全余量。

:::

<ImageView src={require('./img/print-workflow-a-to-e.webp').default} width="70%"/>

点击图像框 (C) 上传要打印的图像。图像上传后，可以移动并缩放它，使其在物体上正确定位。红色轮廓表示根据你先前输入的尺寸生成的物体外形。

如果你计划重复打印此内容，可以选择将它设为 favourite print (D)。这样它会永久保存在打印机内存中，而不会被下一次发送的打印文件覆盖。选择此选项时，需要为该 favourite 提供一个名称，以便之后在打印机上使用该名称识别文件。

接下来，需要将文件发送到打印机。可以发送两种文件：marker files (F) 和 image files (G)。Marker files 会直接打印到食品安全表面上，用于显示物体放置位置。Print files 会在物体放置到表面后打印。如果你已经知道物体应放在哪里，则无需发送 marker file，可以直接发送 image file。

<ImageView src={require('./img/print-workflow-f-to-g.webp').default} width="70%"/>

选择任一文件类型并发送到打印机后，你会看到一个显示上传状态的小弹窗。

<ImageView src={require('./img/uploading-print.webp').default} width="300"/>

### 打印工作流

图像以及可选的 marker files 准备好并发送到打印机后，就可以开始打印了。请按照此流程操作，以确保打印成功。

1. 使用墨盒维护和护理部分中提到的方法擦拭墨盒底部。

    <ImageView src={require('./img/wiping-cartridge.webp').default} width="50%"/>

2. 进入打印机的打印菜单，你会看到每张发送图像最多对应三种不同打印文件。

    <ImageView src={require('./img/print-files-on-printer.webp').default} width="70%"/>

    - `Preparation print`: 此文件会让打印机归位，并将热床移动到可以放置物体的位置。如果你在过去 15 分钟内没有准备过打印机，则在执行第一次 image print 之前必须先运行它。
    - `Marker print`: 此文件会准备打印机，并在食品安全表面上打印标记，用来指示物体的放置位置。
    - `Image print`: 这是最终打印到物体上的打印文件。只有在打印机已经完成准备后才应运行它。

    :::info[Note]

    运行 marker print 时，表面上绝不能放置物体，否则物体会与打印头碰撞。

    :::

3. 如果需要打印 marker file，请点击该选项，并确保右侧所有选项都处于禁用状态：点击它们，并确认任何选项周围都没有绿色轮廓。

    <ImageView src={require('./img/disable-options.webp').default} width="70%"/>

4. 如果你正在打印 marker file 或 preparation file，请趁此机会确认打印表面上完全没有任何物体。确认后，点击打印。

    <ImageView src={require('./img/filament-review-screen.webp').default} width="70%"/>

5. 打印机会执行一些准备步骤，然后打印所选文件。如果在准备过程中任何时候听到 Panda Treat 发出响亮蜂鸣声，这是警告你尚未运行所需的 preparation print file。你应立即停止打印，并运行所需的 preparation file。如果打印机尚未准备好，你会在 Panda Treat 屏幕上看到提示。

    <ImageView src={require('./img/not-prepared-rounded.webp').default} width="120"/>

6. 如果文件成功打印，你将得到以下结果之一：

    - 打印表面上的标记。
    - 处于已准备状态的打印机。
    - 已打印图像的物体。

7. 假设你只是打印了 marker file，那么打印表面上会出现如下所示的红色标记。

    <ImageView src={require('./img/marker-outline.webp').default} width="45%"/>

8. 如下图所示，将物体放入轮廓内。

    <ImageView src={require('./img/object-on-plate.webp').default} width="45%"/>

9. 打印 image file，并观察你的 treat 接收打印图案。

    <ImageView src={require('./img/printed-object.webp').default} width="45%"/>

如果希望颜色更加饱和，请保持 treat 的位置不变，并在打印机屏幕上点击 `Reprint`。

## 打印质量

Panda Treat 打印质量取决于墨盒状态，物体放置，打印机运动以及 ink strip spacing。如果打印质量较差，请先检查简单原因，再更改设置。

- 缺色或断线通常表示墨盒需要擦拭，或之前存储不当。请参阅墨盒护理部分。
- 大幅位置偏差通常表示未先打印标记，或物体在放置后发生了移动。
- 打印走线之间非常明显的间隙或深色线条可能表示 Y 轴运动或 strip spacing 问题。请重新检查皮带张力，然后如有需要，以较小的 `0.01` 增量调整 `Ink Strip Spacing`。请参阅设置部分。
- 如果打印头拖过物品，请停止并重新测量物体高度。再次尝试前，略微增加高度值。
- 可用时，请从首页使用 `Troubleshooting` 进行引导式检查。
- 如果清洁墨盒，准备打印机并确认物体高度正确后问题仍反复出现，请记录清晰的瑕疵照片和所用设置，以便支持人员诊断问题。
