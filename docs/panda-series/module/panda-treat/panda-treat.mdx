## 产品概述

Panda Treat 是一款可快速安装的食用墨水适配器，适用于兼容的 3D 打印机。它可以将标准 3D 打印机转换为食用墨水打印机，用于在饼干，蛋糕，马卡龙，啤酒泡沫，咖啡泡沫以及其他能够安全放置在打印平台上的合适食品或饮品表面打印食品级墨水。

完整系统由 Panda Treat 控制板，食用墨水墨盒，用户自行打印的安装件，打印机接口线缆套件以及引导式本地 Web App 组成。

随附线缆套件包含适用于 A1，P1 和 X1C 类打印机连接的线缆版本。请仅将 Panda Treat 安装到已经提供匹配安装件，固件和官方说明的打印机型号上。请持续关注 Wiki，更多打印机支持会陆续添加。

## 产品特性

- 将兼容的 3D 打印机转换为食用墨水打印机，并配备可快速拆装的工具头附件。
- 快速安装和拆卸让你的打印机既可以用于 3D 打印，也可以用于在可食用物品上打印，无需漫长的安装和拆卸过程。
- 提供引导式本地 Web 界面，用于设置，图像放置与尺寸调整，标记打印以及将文件发送到打印机。
- 支持在适合打印机成型空间的食品和饮品上打印，包括：
    - 饼干
    - 蛋糕
    - 马卡龙
    - 咖啡泡沫和拿铁
    - 啤酒泡沫及其他合适的饮品表面
    - 巧克力及其他平整或轻微弯曲的可食用物品
- 包含标记工作流，用户可以在打印最终图像之前先打印定位引导。
- 包含用于自定义形状的饼干模具生成工作流，可从预览页面导出 STL。
- 支持通过设置页面进行固件和 UI 更新。

## 食品安全

Panda Treat 随附的食用墨水专为食品使用场景生产，并配有产品销售地区所需的相关食品安全文件，MSDS 文件和认证。

请仅使用官方 Panda Treat 食用墨水墨盒和配件。手动填充墨盒很可能导致堵塞并降低打印质量。请勿将普通打印机墨水用于食品应用。

请采用与准备食物时相同的卫生操作。尤其需要注意：

- 不要让食物接触 3D 打印机的任何部位。
- 在打印平台和食物之间放置干净且食品安全的隔离层。
- 将食物放到打印平台之前，请确保它可以安全食用；打印后也要确保它不会接触不安全表面。
- 使用前清洁打印平台区域，并清除任何耗材，灰尘或其他残留物。
- 打印前清洁打印机喷嘴上的任何耗材残留。清除残留物可确保归位准确，有助于避免喷嘴或部件接触可食用表面。
- 墨盒不使用时，请盖上盖子或保持密封。
- 打印前始终确认你正在使用食品安全的墨盒。
