## 产品简介

BIQU Panda Sense Pro 是必趣创新科技（深圳）有限公司推出的一款搭配多款传感器的空气质量检测模块。可以在多场景使用，时时监控空气质量情况

## 技术规格

| Panda Sense Pro | 规格 |
| --- | --- |
| 产品外形尺寸 | 117.12mm x 79.78mm x 73.67mm |
| 电源输入 | USB-C DC5V 2A |
| MCU | ESP32-S3-WROOM-1-N16R8 |
| 电流 | 0.52A（峰值0.64A） |
| 逻辑电压 | 3.3V |
| 扩展功能及接口 | I2C接口(可接Panda Sense模块)，重置按键(可恢复出厂设置) |

</div>

## 产品特点

- 功能大幅度提升，不仅能测试温湿度，还可以检测PM2.5，CO2，甲醛，TVOC等

- 带有3.5寸RGB屏幕显示，可直观看清各个参数值

- 设有物理按键，能够灵活切换显示页面

- 本产品板载WIFI模块，可以通过网络进行信息交互

- 预留重置按键，一键恢复出厂设置

- 预留硬件烧录口，当OTA升级失败时可以通过硬件烧录口二次拯救

## 使用指南

### 首次使用

1. 连接电源：使用 USB-C 线为设备供电，屏幕将自动亮起

2. 等待预热：eTVOC 模块需要预热 5 分钟，请等待预热完成后再查看 eTVOC 数据

3. 配网设置：根据屏幕显示的热点名称在手机或电脑连接后进行配网操作

### 配网

没有配置网络时，显示屏会显示热点名称"Panda_Sense_Pro_XXXXXXXXXXXX"

<ImageView src={require('./img/ap-name.webp').default} width="256"/>

使用手机或者电脑连接该热点，密码默认是 `987654321`

<ImageView src={require('./img/ap-select.webp').default} width="256"/>

连接热点后如果没有自动跳转到 WEB UI，需要在浏览器手动输入: `http://192.168.254.1/`

选择语言。

<ImageView src={require('./img/select-lang.webp').default} width="256"/>

扫描 WiFi 后，选择需要连接的 WiFi 名称，输入 WiFi 密码后点击连接

<ImageView src={require('./img/ap-connect.webp').default} width="256"/>

### 选择时区

## 功能介绍

### 查看传感器数据

#### 实时数据

有多种查看数据的方式：

- 显示屏

    <ImageView src={require('./img/sensor-data-screen.webp').default} width="256"/>

- WEB UI

    <ImageView src={require('./img/sensor-data-web.webp').default} width="256"/>

- Klipper 虚拟传感器

    <ImageView src={require('./img/sensor-data-klipper.webp').default} width="256"/>

    :::info[注意]

    该功能需要修改 moonraker.conf 相关配置来实现，如果不显示数据，则需要在 printer.cfg 中至少添加一种设备，比如风扇

    <ImageView src={require('./img/fan.webp').default} width="256"/>

    :::

    - moonraker.conf
        ~~~
        [mqtt]
        address: xxx.xxx.xxx.xxx #Change to match the IP address of Panda Sense
        port: 1883
        client_id: moonraker
        enable_tls: False
        mqtt_protocol: v3.1
        enable_moonraker_api: False
        instance_name: my_printer

        [sensor panda_sense_pro]
        type: mqtt
        name: Panda Sense Pro #Change if you have more than 1 Sense Pro to tell them apart
        qos: 0
        state_topic: pandasensepro/sensor
        state_response_template: |
        {% set resp = payload|fromjson %}
        {% set edata = resp["data"] %}
        {set_result("temperature", edata["temper"]|float)}
        {set_result("humidity", edata["humidity"])}
        {set_result("pm2_5", edata["pm2.5"])}
        {set_result("pm10", edata["pm10"])}
        {set_result("co2", edata["co2"])}
        {set_result("etvoc", edata["etvoc"])}
        {set_result("ch2o", edata["ch2o"])}
        {set_result("aqi", edata["aqi"])}

        parameter_temperature:
        units=℃
        parameter_humidity:
        units=%
        parameter_pm2_5:
        units=µg/m³
        parameter_pm10:
        units=µg/m³
        parameter_co2:
        units=ppm
        parameter_etvoc:
        units=mg/m³
        parameter_ch2o:
        units=mg/m³
        parameter_aqi:
        units=-
        history_field_humidity_consumption:
        parameter=humidity
        desc=Printer humidity consumption
        strategy=average
        units=%
        init_tracker=true
        precision=6
        exclude_paused=false
        report_total=true
        report_maximum=true
        ~~~

#### flash_download_tool (Type-C 线刷)

:::info[注意]

Panda Sense Pro 理应可以自由 OTA，此步骤仅在无法 OTA 时使用

:::

- 如果电脑上没有 CH340 的驱动，请先下载安装驱动 [CH341SER.EXE](https://www.wch.cn/downloads/CH341SER_EXE.html)

- 下载 [Flash 下载工具](https://www.espressif.com/zh-hans/support/download/other-tools)

    <ImageView src={require('./img/download_esp_tool.webp').default} width="70%"/>

- 下载 [Panda Sense Pro 可烧录固件](https://github.com/bigtreetech/PandaSensePro/tree/master/recovery_tool/flash)

- 通过 `Type-C` 将 Panda Sense Pro 插到电脑上，电脑的设备管理器中理应识别出一个新的 COM 端口

    <ImageView src={require('./img/typec.webp').default} width="35%"/>

- 打开 `flash_download_tool_3.x.x.exe`，在弹窗中按照下图配置：

    ChipType: `ESP32-S3`

    WorkMode: `Develop`

    LoadMode: `UART`

    <ImageView src={require('./img/download.webp').default} width="200"/>

- 烧录软件的配置如下图

    <ImageView src={require('./img/esp_tool_config.webp').default} width="400"/>

    1. 设置 `.bin` 文件的烧录地址，并且前面的复选框都勾选上

        - `panda_sense_pro_production.bin` 写入到 `0x0000`

    2. 设置 COM 端口为实际的端口（可在电脑的 `设备管理器`->`端口` 中查看），并且设置一个合适的波特率，推荐使用 `460800`

    3. 点击 `START` 开始烧录，等待烧录完成，断电重启即可。
