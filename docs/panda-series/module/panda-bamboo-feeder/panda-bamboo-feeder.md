## 产品简介

BIQU Panda Bamboo Feeder（熊猫喂竹器）是一款专为 FDM 3D 打印机设计的智能送料缓冲系统，搭载ESP32主控芯片，支持WiFi无线连接，集主动正压送料缓冲,自动进退料,智能续料,多台联动等功能于一体。

无论是软性耗材,大卷耗材,长距离路径，还是需要长时间无人值守的批量打印任务，Panda Bamboo Feeder都 能稳定应对，让打印更省心,更高效。

并为拓竹 X1 / P1 / P2 / A1 系列提供专属优化与专用供电接口，适配更省心，体验更完整。

## 产品特点

- **主动正压送料缓冲：** 软料,脆料,5kg+ 大卷料，无论什么耗材,多复杂的路径，全程稳定正压，大幅减少挤出不足与拉丝断层。打印稳定，从送料开始。

- **自动进退料：** 耗材插入，一键启动，自动推送至打印头，同步发送进料指令至打印机；按下退料键，耗材同步退回入料口。

- **智能续料：** 单个支持双盘自动切换，最多三台联动,四盘接力。大型模型,批量任务，从头跑到尾，交给它。

- **WiFi 远程控制：** 内置 ESP32 主控，支持 WiFi 连接，Web UI 网页端与 BQ App 双端操控。

- **注塑精品外壳：** 注塑精密成型，半透明黑色外壳隐约透出内部构造，严丝合缝，质感出众。

- **广泛兼容 FDM 打印机：** 兼容绝大多数市面主流 FDM 打印机，并为 Bambu Lab X1 / P1 / P2 / A1 系列提供专属优化，开箱即用。
