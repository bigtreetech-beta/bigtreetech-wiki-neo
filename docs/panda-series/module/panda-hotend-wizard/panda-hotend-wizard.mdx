## 产品简介

BIQU Panda Hotend Wizard 是深圳必趣科技有限公司推出的一款用于快速维修拓竹适配热端堵头的维修工具。用户可通过旋钮和按键进行操控，为堵头的喷嘴提供良好的维修环境，提高喷嘴的利用率。目前适配的机型包括拓竹 A1，A1 mini，P1S，X1C，P2S，H2D，H2S 等。暂时不支持 H2C-Voteck 系统热端。

## 产品特点

- 产品整体小巧轻便，轻松放置，不占空间。
- 配备总电源开关，可以方便快速切断总电源，保障操作安全。
- 旋钮快速设置目标温度，按键开关风扇，操作直观。
- 多机型适配，支持 A1，A1 mini，P1S，X1C，P2S，H2D，H2S等拓竹主流机型。
- 最高支持 $290^\circ C$ 的加热温度，为多种特殊耗材提供有利的维修环境，提高维修质量。
- 配送专用通针，方便用户快速疏通堵头。
- 数码管显示温度清晰明了，配合状态灯实时展现加热棒和风扇工作状态。

## 使用指南

### P1 / X1 系列维修步骤

1. 关闭打印机电源并拆下工具头前盖：确保打印机完全断电，拆下工具头前盖，露出热端组件。
2. 连接维修夹具与接口：使用包装内附带的 P1/X1 系列维修夹具固定热端，拧紧螺丝，并将热端的加热和风扇接口连接至设备机身的 P1/X1 连接接口。注意：机身加热块的接口要拔掉。

    <ImageView src={require('./img/p1x1.webp').default} width="60%"/>

3. 设置目标温度：打开产品电源，通过旋钮设置目标温度，建议从较低温度开始测试并逐步升高。
4. 疏通堵头：等待温度稳定后，使用包装内附带的专用通针进行疏通清理。

### H2 系列 / P2S 维修步骤

1. 关闭打印机电源并拆下工具头前盖：确保打印机完全断电，拆下工具头前盖。
2. 连接维修夹具与接口：使用包装内附带的 H2/P2 系列维修夹具固定热端，快拆的维修夹具自带磁铁，可以吸紧热端，热端安装上去后，请扣上保护罩，并将机身自带的加热块接口连接至设备机身的快拆热端连接接口。

    <ImageView src={require('./img/p2sh2.webp').default} width="60%"/>

3. 设置目标温度并维修。开机后通过旋钮设定目标温度，升温稳定后进行堵头疏通。

### A 系列维修步骤

1. 步骤与P2S和H2系列快拆热端相似，关闭打印机电源并做好防护。确保打印机断电，准备好维修夹具与通针。
2. 连接快拆热端。使用 A1 系列维修夹具固定热端，并将机身自带的加热块接口连接至设备机身的快拆热端连接接口。
3. 加热与疏通。设置合适的目标温度，待数码管显示温度达到设定值后，使用通针清理堵头。

## 功能介绍

<ImageView src={require('./img/fuction.webp').default} width="60%"/>

1. 总电源开关

    用于开启和关闭整机电源。维修完成后请务必关闭总电源，避免长时间待机。

2. 风扇开关

    用于开启和关闭风扇。在加热维修过程中，可根据需要开启风扇对热端进行散热或冷却。

3. 旋钮功能

    旋钮是主要的交互方式，支持设置目标温度，快捷温度，关闭加热，设置热端类型等操作。

    - 设置目标温度

        1. 短按旋钮进入设置模式，此时数值会闪烁。
        2. 旋转旋钮调节目标温度值，步长为 $5^\circ C$，顺时针增大，逆时针减小。
        3. 再次短按按钮确认温度，并退出设置模式。

    - 快捷温度

        在设置目标温度时，双击旋钮，将温度快速调节为 $250^\circ C$。再次短按旋钮确认温度值。

    - 关闭加热

        非设置模式下，长按旋钮后关闭加热输出。

    - 设置热端类型

        目标温度为 $0^\circ C$ 时，长按旋钮进入热端类型选择。此时蓝色数码管闪烁，转动旋钮选择类型后，短按旋钮确认类型。建议选择数值 1，自动识别。如果出现温度无法上升或温度不稳定的情况，可手动设置为对应热端。

        | 数值 | 热端类型 |
        | --- | --- |
        | 1 | 自动识别热端 |
        | 2 | 自带加热模块，使用设备自身加热棒时的 PID 参数 |
        | 3 | 拓竹热端，使用拓竹原装热端时的 PID 参数 |

4. 数码管显示

    机身配备双数码管，分工明确。

    - 上方数码管

        红色数码管，显示当前实际温度或当前菜单名称。

    - 下方数码管

        蓝色数码管，显示目标温度或当前选中的热端类型。

- 自动关闭加热

    当用户连续 20 分钟没有操作设备时，系统将自动关闭加热输出，防止干烧及意外过热。

## 故障排除

### 加热温度无法上升或温度不稳定

1. 需要手动切换 PID 参数。

    - 使用自带加热棒维修时，选择数值 2。
    - 使用拓竹原装热端维修时，选择数值 3。

2. 如果是快拆热端，检查快拆热端是否跟加热块接触良好，不要有明显间隙。如下图箭头指示处。
