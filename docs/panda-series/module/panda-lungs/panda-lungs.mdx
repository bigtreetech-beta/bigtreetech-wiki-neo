## 产品简介

**Panda Lungs** 是一款专为 Bambu Lab P1/X1 系列打印机配套设计的空气净化辅助设备,主要功能包括:

- **自动控制风扇启停**:根据打印机状态(打印中/暂停/完成/失败)自动启停风扇。
- **温湿度显示**:通过 TM1650 数码管实时显示环境温度,湿度。
- **TVOC 检测**:实时检测挥发性有机物(TVOC)浓度并显示。
- **蜂鸣器报警**:TVOC 超标时自动发出报警提示音。
- **Web UI 远程控制**:通过浏览器访问 Web UI, 远程查看状态和配置参数。
- **参数记忆**:所有配置断电后自动保存。

### 控制面板功能

控制面板是核心配置页面,包含以下功能:

#### 打印机暂停 — 延时关风扇

打印机进入**暂停**状态后,风扇保持运行,倒计时结束后自动关闭。

- 范围:**0 ~ 60 分钟**
- 默认值:**5 分钟**
- 操作:点击 `+` / `-` 调整；长按可快速步进(± 5 分钟)
- 设为 **0 分钟** 时,暂停后立即关闭风扇

#### 打印机完成 / 失败 — 延时关风扇

打印机进入**完成**或**失败**状态后,风扇保持运行,倒计时结束后自动关闭。

-范围:**0 ~ 60 分钟**
-默认值:**5 分钟**
-操作:点击 `+` / `-` 调整；长按可快速步进(± 5 分钟)
-设为 **0 分钟** 时,状态切换后立即关闭风扇

#### 温度单位

切换数码管温度显示单位:

| 选项               | 说明           |
| ------------------ | -------------- |
| **Celsius / ℃**    | 摄氏度(默认) |
| **Fahrenheit / ℉** | 华氏度         |

#### TVOC 报警开关

| 状态         | 说明                                         |
| ------------ | -------------------------------------------- |
| 开启(默认) | TVOC 超过阈值(0.6 mg/m³)时,蜂鸣器自动报警 |
| 关闭         | 禁用蜂鸣器报警                               |

#### 保存设置

点击页面底部红色 **保存设置** 按钮,将所有配置写入设备存储,断电后不丢失。

> **重要:每次修改参数后必须点击「保存设置」才会持久化！**

#### 按键操作说明

设备面板上配有一个多功能按键(IO9),支持以下操作:

| 操作                              | 功能                                    |
| --------------------------------- | --------------------------------------- |
| **单击**                          | 手动切换风扇开 / 关(进入手动控制模式) |
| **三连击**                        | 进入工厂测试模式(设备重启后进入)      |
| **长按(约 2 秒)**(测试模式下) | 退出工厂测试模式,恢复正常模式          |

#### 风扇自动控制逻辑

设备根据打印机实时状态自动控制风扇,规则如下:

| 打印机状态        | 风扇行为                                 |
| ----------------- | ---------------------------------------- |
| **打印中**        | 立即开启风扇                             |
| **暂停**          | 保持开启,倒计时(可配置)结束后自动关闭 |
| **打印完成**      | 保持开启,倒计时(可配置)结束后自动关闭 |
| **打印失败**      | 保持开启,倒计时(可配置)结束后自动关闭 |
| **空闲 / 未连接** | 立即关闭风扇                             |

#### 手动模式

- **单击按键**:立即切换风扇开 / 关,进入手动模式
- 手动模式下自动控制逻辑暂停

#### 数码管显示说明

设备配备 3 组 TM1650 数码管,各组独立显示以下内容:

#### TVOC 数码管(上侧)

| 显示内容                   | 说明                                    |
| -------------------------- | --------------------------------------- |
| 数字(如 `0.15`)          | 传感器正常,显示实时 TVOC 浓度(mg/m³) |
| 倒计时数字(如 `420`→`0`) | 传感器预热中,等待就绪(最长 10 分钟)  |
| `E.00`                     | 传感器超时无响应,请检查硬件连接        |

#### 温度数码管(中间)

| 显示内容          | 说明                                 |
| ----------------- | ------------------------------------ |
| 数字(如 `25.6`) | 实时温度值(℃ 或 ℉,由控制面板设置) |
| `- -`             | 传感器读取失败                       |

#### 湿度数码管(下侧)

| 显示内容        | 说明                |
| --------------- | ------------------- |
| 数字(如 `48`) | 实时相对湿度(%RH) |
| `- -`           | 传感器读取失败      |

#### 蜂鸣器报警说明

当 **TVOC ≥ 0.6 mg/m³** 时,蜂鸣器自动按以下节奏持续报警:

```text
响 1 秒 → 停 0.3 秒 → 响 1 秒 → 停 0.3 秒 → ...(循环)
```

- 可在控制面板中关闭报警(TVOC 报警开关)
- 关闭状态断电后保持,无需每次重新设置

## 常见问题 FAQ

### Q1:设备上电后找不到 AP 热点?

**解决方法:**

- 等待约 10 秒,设备启动需要时间
- 确认设备 USB-C 供电正常(5V/1A 以上)
- 尝试重新上电

### Q2:Wi-Fi 连接失败或一直显示"连接中"?

**解决方法:**

- 确认 Wi-Fi 密码输入正确
- 确认路由器支持 2.4GHz(不支持纯 5GHz)
- 如果多次失败,尝试恢复出厂设置后重新配置

### Q3:无法扫描到打印机?

**解决方法:**

- 确认 Panda Lungs 和打印机在**同一局域网**内
- 确认打印机已开机且联网
- 尝试手动输入打印机 IP 和 SN 码进行绑定
- 检查打印机 Access Code 是否正确(在打印机设置中查看)

### Q4:参数设置后重启丢失?

**解决方法:**

- 确认修改后点击了 **保存设置** 按钮

### Q5:数码管不显示或显示异常?

**可能原因:**

- 传感器硬件连接问题
- I2C 通信不稳定

**解决方法:**

- 检查传感器连接
- 重新上电

### Q6:如何恢复出厂设置?

1. 进入 **设置** 页面
2. 点击 **Factory Reset(恢复出厂设置)** 按钮
3. 确认后设备自动重启并清除所有配置

> 恢复出厂设置会清除所有配置(Wi-Fi,打印机绑定,参数设置等),请谨慎操作！

### Q7:如何通过主机名访问 Web UI?

默认主机名为 `PandaLungs`,在浏览器输入:

```
http://PandaLungs.local
```

如果无法访问,可以在 **Wi-Fi & IP** 页面查看设备 IP 地址,直接使用 IP 访问。
