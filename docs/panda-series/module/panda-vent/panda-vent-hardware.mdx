# Panda Vent 安装说明

## 安装前准备

:::tip

请根据不同机型下载对应的外壳文件，并按照文件要求打印外壳。

:::

:::info[模型下载]

Makerworld

- [Panda Vent Smart Riser for P1 X1](https://makerworld.com/zh/models/2854242-panda-vent-smart-riser-for-p1-x1)
- [Panda Vent Smart Riser for P2 X2](https://makerworld.com/zh/models/2854250-panda-vent-smart-riser-for-p2-x2)

Github Repo [bigtreetech/Panda-Vent](https://github.com/bigtreetech/Panda-Vent)

:::

### 所需工具及材料

| 名称         | 规格 / 数量      | 说明                            |
| ------------ | ---------------- | ------------------------------- |
| 自攻螺丝     | `CM3x6` x 41 pcs | 用于固定各打印件PCB板及装饰盖板 |
| 内六角螺丝刀 | `2.0`            | 用户自备                        |

:::warning

安装前请确认所有打印件方向接口位置和螺丝孔位一致，避免强行装配导致打印件损坏。

:::

## 打印件拼接步骤

### 组装打印件 ABC

- 将 `打印件A` `打印件B` 和 `打印件C` 按图示位置对齐。
- 使用 `CM3*6 自攻螺丝` 从上方锁紧固定。
- 确认各卡扣定位柱和边框贴合到位。

<ImageView
    src={require('./img/panda-vent-install-step1.webp').default}
    width="60%"
/>

### 组装打印件 CDE

- 将已组装的侧边结构与 `打印件D` `打印件E` 对齐。
- 按图示方向装入 `打印件CE`，并使用 `CM3*6 自攻螺丝` 固定。
- 锁紧后检查左右侧边是否保持平行，避免影响后续四面拼接。

<ImageView
    src={require('./img/panda-vent-install-step2.webp').default}
    width="60%"
/>

## 前框打印件组装

### GH 打印件组装

- 将 `打印件G` 与 `打印件H` 按图示方向拼接。
- 使用 `CM3*6 自攻螺丝` 从连接位置固定。
- 固定后确认中间拼接处无明显错位。

<ImageView
    src={require('./img/panda-vent-install-step3.webp').default}
    width="60%"
/>

### 前脸装饰打印件组装

- 将左右前脸装饰打印件安装到前框两侧对应位置。
- 按图示螺丝孔位使用 `CM3*6 自攻螺丝` 固定。
- 安装完成后确认装饰件外观面贴合平整。

<ImageView
    src={require('./img/panda-vent-install-step4.webp').default}
    width="70%"
/>

### 按键开关及 FGHI 打印件组装

- 将按键开关装入右装饰件对应孔位。
- 将 `打印件F` `打印件G` `打印件H` `打印件I` 按图示位置组合。
- 使用 `CM3*6 自攻螺丝` 固定连接位置。
- 安装后轻按按键，确认按键动作顺畅无卡滞。

<ImageView
    src={require('./img/panda-vent-install-step5.webp').default}
    width="70%"
/>

### 灯罩及盖板组装

- 将灯罩与对应盖板按图示方向装入前框。
- 将连接线按图示槽位放置，避免被压住或拉扯。
- 装配完成后检查灯罩是否完全嵌入到位。

<ImageView
    src={require('./img/panda-vent-install-step6.webp').default}
    width="70%"
/>

## 四面打印件拼接

- 将前框左右侧框和后侧打印件围合成完整外框。
- 对齐四角连接位置后，使用 `CM3*6 自攻螺丝` 锁紧固定。
- 拼接完成后检查外框是否平整，四角是否贴合。

<ImageView
    src={require('./img/panda-vent-install-step7.webp').default}
    width="70%"
/>

## 安装风门套件

- 将风门套件放入外框内部对应安装位置。
- 按图示方向固定风门组件及相关连接件。
- 确认风门开合顺畅，不与外壳或线材发生干涉。

<ImageView
    src={require('./img/panda-vent-install-step8.webp').default}
    width="70%"
/>

## PCB 板及线材装饰盖板装配

- 将 PCB 板安装到外框内部指定位置。
- 使用 `CM3*6 自攻螺丝` 固定 PCB 板。
- 将线材整理到对应走线槽内，再安装线材装饰盖板。
- 装配完成后检查线材是否被压住，接口是否牢固连接。

<ImageView
    src={require('./img/panda-vent-install-step9.webp').default}
    width="70%"
/>

## 安装到打印机顶部槽位

- 从打印机上取下玻璃顶盖。
- 将 Panda Vent 与打印机顶部对齐，然后从上方放置到打印机顶部。
- 将玻璃顶盖放回 Panda Vent 上方，然后检查 Panda Vent 是否放置平整且稳定。

<ImageView
    src={require('./img/panda-vent-install-step10.webp').default}
    width="70%"
/>

## 安装检查

- 所有 `CM3*6 自攻螺丝` 已锁紧，无漏装或松动。
- 四面外框拼接平整，连接处无明显缝隙或错位。
- 按键开关按压顺畅，无卡滞。
- 灯罩盖板风门套件和 PCB 板均安装到位。
- 线材走线整齐，没有被外壳盖板或风门压住。
- 整体放入打印机顶部后，位置稳定，不影响打印机开盖或其他结构动作。

:::tip

完成结构安装后，可继续参考 Panda Vent 软件说明书完成首次连接 WiFi 配置和打印机绑定。

:::
