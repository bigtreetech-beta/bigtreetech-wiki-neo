# Panda Vent

## 产品简介

**Panda Vent** 是一款专为 FDM 3D 打印机设计的智能通风增高架。它可以实时监测打印状态，并自动控制风门和灯光，让你的打印机更加智能 / 安静 / 清凉。

## 核心功能

- RGB 状态灯实时显示打印机状态，包括空闲 / 准备中 / 打印中 / 暂停 / 已完成和错误状态，一眼即可掌握当前打印状态。
- 智能风门控制，支持 **自动模式** 和 **手动模式**。自动模式会根据打印机状态自动控制风门；手动模式下可通过按键切换风门开关。
- 增强散热优化内部气流并主动排出热空气，有效帮助防止因腔体温度过高导致的喷嘴堵塞，并提升长时间打印的稳定性。
- 增高架设计抬高顶部玻璃或 AMS 安装空间，显著减少 PTFE 管与顶部玻璃盖之间的摩擦，有助于降低管材磨损以及摩擦噪音和打印失败风险。
- 通过本地 MQTT 协议与 Bambu Lab 打印机通信。无需云端连接，响应快速且稳定可靠。
- 内置 Web UI 可通过任意浏览器配置 Wi-Fi / MQTT / 灯光 / 风门及其他功能。
- 支持通过 Web UI 进行固件升级，维护简单方便。
- **自动识别 LED 灯带：**
    - 可自动检测连接的是 1 组还是 2 组 LED 灯带：
    - **1 组灯带：** 16 颗 LED
    - **2 组灯带：** 27 颗 LED
    - 无需手动设置，即插即用。

## 适用场景

Panda Vent 适合以下用户：

- 需要更好的腔体散热或气流改善。
- 希望通过灯光直观查看打印机状态，而无需频繁查看打印机屏幕。
- 喜欢 DIY 智能外设，并希望将设备集成到现有 **Home Assistant** 或 **MQTT** 系统中。
- 想要优化 PTFE 管走线，减少摩擦噪音，并降低管材磨损。

## 硬件功能

### 按键功能

| 按键                   | 位置                                                     | 功能                                                |
| ---------------------- | -------------------------------------------------------- | --------------------------------------------------- |
| **带灯按键** (GPIO 12) | 设备面板                                                 | 单击：切换风门开/关；长按（3秒）：切换自动/手动模式 |
| 手动模式               | 处于自动模式下，单击按钮会回到手动模式，并且立刻翻转扇叶 | 处于手动模式下，单机立刻翻转扇叶                    |
| BOOT 按键 (GPIO 0)     | 设备侧面                                                 | 长按（3秒）：恢复出厂设置                           |

### 指示灯

| 状态         | LED 表现              |
| ------------ | --------------------- |
| **自动模式** | LED 常灭              |
| **手动模式** | LED 闪烁（0.5秒间隔） |

### 接口与灯组

- 支持 **1 组灯带**（16 颗 LED）或 **2 组灯带**（27 颗 LED）
- 设备会通过 ADC 电压自动检测当前插入的灯组数量，无需手动配置
