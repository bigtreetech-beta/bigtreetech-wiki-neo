### 40 pin GPIO

<ImageView src={require('./img/pi2-gpio.webp').default} width="80%"/>

GPIO 引脚的计算方式如下：

GPIO4_B2 = (‘B’ - ‘A’) * 8 + 2 = 1 * 8 + 2 = gpiochip4/gpio10

GPIO3_D7 = (‘D’ - ‘A’) * 8 + 7 = 3 * 8 + 7 = gpiochip3/gpio31
