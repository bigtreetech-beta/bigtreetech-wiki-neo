# BIGTREETECH Pad5 V2

<div class="div-table">

## 简介

BIGTREETECH Pad5 V2.0是深圳必趣创新科技有限公司推出的一款搭配树莓派CM5使用的拓展屏板，让客户对CM5的功能使用更加方便。

## 产品特点

1. 同时兼容CM5/CM4/CB2/CB1等多个SOC核心板；

2. 板载USB-OTG，方便客户对EMMC版本的核心板进行镜像烧录；

3. 使用CM5时支持USB3.0接口通讯；

4. 预留XH2.54 2P的CAN接口，方便客户DIY使用；

5. 5寸IPS电容屏，显示效果更佳，使用体验更好；

6. 各个接口增设ESD保护，提高产品稳定性；

7. 使用CM5时支持一键开关机，方便客户使用；

8. 板载上位机USB-HUB，一路通过CH340E连接到UART端口，一路直连USB-OTG，方便客户烧录镜像和调试Soc；

9. 多种USB接口方式，方便客户DIY。
