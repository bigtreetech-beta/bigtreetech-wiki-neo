## 产品简介

BIGTREETECH CB1 是针对树莓派 CM4 缺货问题推出的替代方案,所有输出信号采用 2 个 100PIN 微型 BTB 高速连接座与外面扩展底板方便快速连接(包括百兆以太 HDMI 等)，板载 2.4G WIFI。

## 功能亮点

- CPU: 全志 H616，四核 Cortex-A53 @1.5GHz
- GPU: Mali G31 MP2, 支持 OpenGL3.2
- RAM: 512MB/1GB DDR3L SDRAM
- 显示: HDMI2.0A 接口，支持 4K 显示器
- USB2.0 端口
- 百兆以太网+百兆 WIFI
- 与树莓派 CM4 完全相同的 BTB 座子
