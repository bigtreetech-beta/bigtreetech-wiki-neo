## 接口介绍

### USB 供电

主板上电之后， 电源灯会亮起，表示供电正常。板上标识 VUSB 是电源选择端，仅当使用USB 给主板供电时，才需要使用跳帽将 VUSB 短接。

<ImageView src={require('./img/mmb-can-usb.webp').default} />

### Servo 接线

:::warning[警告]
此处演示为使用MMB的板载DC-DC为舵机供电。

MMB可以胜任这项工作，但如果舵机出现损坏，将迫使MMB的5V供电触发短路保护，致使MCU离线。
:::
