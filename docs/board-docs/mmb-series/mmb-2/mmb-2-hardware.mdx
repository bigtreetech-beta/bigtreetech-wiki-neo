## 硬件功能配置

### USB 供电

主板上电之后， 电源灯会亮起，表示供电正常。板上标识VUSB是电源选择端，仅当使用USB给主板供电时，才需要使用跳帽将VUSB短接。

<ImageView
	src={require('./img/mmb-can-v2-0-usb.webp').default}
	alt="mmb can v2"
	width="80%"
/>

### 舵机接口

:::info[注意]
此处演示为使用MMB的板载DC-DC为舵机供电。

MMB CAN V2 在舵机输出口添加了一枚3A的自恢复保险丝用以避免电源触发短路保护，足以胜任该应用。但需确保其接入MOT1或MOT2连接器，且尽量避免同步动作，或开启`servo_always_active: 1`来允许舵机重启后仍有持续的控制信号。
:::
